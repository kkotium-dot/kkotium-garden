# DECISION — 크롭 주제 완전포함 원칙 (2026-06-07)

> 표준 보강: docs/design/THUMBNAIL_CROP_EDIT_STANDARD.md §1 "부분 잘림 반려"의 정량 규칙화.
> 트리거: 명화 대표 B 크롭에서 유리병 하단이 프레임에 과도하게 붙어 "잘린 느낌" — 운영자 지적.
> 비가역 0(설계/표준 기록). 환경분담(#41): 본 문서=Desktop 직접 기록, 로직 구현=Code(T6).

## 1. 원칙 (전상품 · 자동+수동 공통)
- 썸네일/대표 크롭은 **주제(제품)의 바운딩박스를 100% 포함 + 최소 패딩**해야 한다. 제품 본체 일부라도 프레임에 잘리면 반려.
- "제품"의 정의 = 판매 단위 본체(예: 디퓨저 = 유리병 + 우드캡/리드 스틱). 연출 소품(올리브잎 등)은 잘려도 허용.
- 최소 패딩 = 주제 bbox 기준 각 변 >= 8% (권장 10~12%). 특히 바닥/상단 잘림 0.

## 2. 자동 크롭 (auto candidate)
- 피사체 검출(subject bbox) -> 정사각 후보는 **bbox + 패딩을 완전 포함하는 최소 변**으로 산출.
- 변이 소스 한계로 부족하면 우선순위:
  1) canvas_expand(outpaint)로 배경 생성 확장(무업스케일, 라인 A/B 공통)
  2) 업스케일(<=2x 권장)
  3) 색 매칭 패드(최후)
- bbox가 정사각에 안 들어오면 절대 "잘라서" 맞추지 말 것 — 확장으로 해결.

## 3. 수동 크롭 (operator drag)
- 운영자가 그린 박스가 검출된 subject bbox를 침범(클리핑)하면:
  - 경고 표시("제품이 잘립니다") + 1클릭 "주제 완전포함으로 맞춤"(박스를 bbox+패딩으로 스냅).
  - 침범 상태로는 적용 차단(confirm 불가). 단, 연출 소품만 걸친 경우는 허용(판정 기준 = 주제 bbox).

## 4. 명화 적용
- 최종 대표 B = 가죽 도어 단품, 유리병 전체 포함(바닥 여유), 라벨 보존, 1:1 1000px — 본 원칙 충족.
- 최고화질 옵션: canvas_expand로 무업스케일 마감 가능(운영자 승인 시 Adobe 커넥터).

## 5. Code 반영 (T6 — feature/crop-studio 동일 브랜치)
- simple-crop 후보 생성 + 크롭 스튜디오(T1) 수동 박스에 subject-containment 가드 추가.
- quality-classifier의 subject 영역 산출을 bbox 추정에 재사용(이미 subject 지표 존재).
- 표준 문서 §1 "부분 잘림 반려"에 본 정량 규칙(>=8% 패딩 · 연출소품 예외 · 확장 우선) 명문화.
