### ★자산 보호 — 삭제 vs 대체 소싱 (Cowork, `09fab39` 배포·prod검증)

**운영자 질문**: "품절 상품은 명화처럼 삭제가 좋은가, 보관이 좋은가?"

**전문가 판단(파워셀러 실무 표준)**: **품절은 보관, 소싱 단절은 조건부**.
네이버 스토어의 실질 자산은 상품 URL에 누적된 이력 — 리뷰·구매평, 판매 건수,
찜, 상품ID 기준 검색 순위. 삭제하면 전부 영구 소멸하고 재등록해도 리뷰 0인
신규 상품이 되어 순위 회복에 수개월. 유료 셀러툴들도 삭제 대신
"판매중지 + 대체 소싱"(같은 URL에서 공급처만 교체 → 순위·리뷰 승계)을 민다.

| 상황 | 권장 |
|---|---|
| 일시 품절 | 품절 처리(OUTOFSTOCK) — 재입고 시 즉시 부활 |
| 장기 품절 | 판매중지(SUSPENSION) — 검색에서 내려가나 URL·리뷰 보존 |
| 소싱 단절 + **판매 이력 O** | **판매중지 + 대체 소싱** (삭제 금지 — 자산 소멸) |
| 소싱 단절 + **판매 이력 X** | 삭제 안전 (지킬 자산 없음 — 명화 케이스) |

**앱 반영(전상품 범용 #55)**:

- 배지: 자산 있으면 "삭제 권장" → **"대체 소싱 권장"**으로 문구·툴팁·꼬띠 보이스 분기
