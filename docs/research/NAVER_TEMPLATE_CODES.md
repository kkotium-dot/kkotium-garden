## 사용 가이드

- **API 자동발행 (꽃틔움 가든 앱)**: 정보고시는 ETC 인라인 자동 생성. 템플릿 번호 입력 불필요.
- **엑셀 수동등록**: 정보고시 칸에 템플릿 코드 2976841 입력.
- **신규 템플릿 추가 시**: 이 문서에 코드 번호를 추가 기록 (앱 연동은 배송 템플릿만).
