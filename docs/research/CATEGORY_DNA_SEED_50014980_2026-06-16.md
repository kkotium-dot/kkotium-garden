## 전상품 공통 적용 메모

- 본 카드는 차량용방향제(50014980) 인스턴스 — 방법은 전 카테고리 동일(find_category -> datalab 3종 -> search_shop -> 합성).
- Stage 1에서 CategoryDna 테이블에 로드. parentDnaId로 자동차용품 상위 카드 상속 가능.
- 슬롯 시퀀스/필수슬롯은 결정테이블 입력 -> SlotPlan 생성.
- 명화 cut-1~4 = scent_note 슬롯 배경으로 확정(썸네일/히어로 아님 — 운영자 평가 정합).
- 시즌 시그널(여름 피크) -> 4~5월 이미지·발행 준비 완료 목표 = ROI 타이밍.
- 엔진 로깅 모델은 SlotGeneration(table slot_generation) — 6축 generation 테이블 충돌 회피(Code 판단·정합).
