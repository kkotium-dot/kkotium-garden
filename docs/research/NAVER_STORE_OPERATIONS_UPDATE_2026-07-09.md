## 1. 커머스 API (주문·상품 연동 직접)
================================================================
- 인증 OAuth2(client_credentials·bcrypt 서명·ms timestamp)·내스토어=SELF. 인증 만료 시 자동 휴면. IP 등록(최대 3개·유동IP는 매일 갱신 필요).
- **Rate limit: 내스토어 전 API 2RPS 고정·429 시 백오프. 헤더 GNCP-GW-RateLimit-Remaining 모니터링.** 대량은 Queue+스케줄러.
- 주문 수집: last-changed-statuses(lastChangedFrom/To·+09:00·%2B). 조회 좁으면 발송/클레임 누락 → 전 상태 배치+overlap.
- **★단위가격 unitCapacity(2026)**: {unitPriceYn, totalCapacityValue, unitCapacity, indicationUnit}. 대상 카테고리인데 필드 없거나 null이면 **등록/수정 불가**(4-29~). 
- **★(v2)채널 상품 수정 함정**: detailContent·seoInfo(태그)를 null로 보내면 초기화됨(과거엔 유지). 유지 필드 명시 재전송 필수.
- 조합형 옵션 최대 3개·조합 재고 합=상품 재고. 표준형 색상 세분화 제한.
- **★리뷰 조회 API 미제공**(공식) — 리뷰 수집 불가·유도만 가능. 크롤링 약관 위반.
