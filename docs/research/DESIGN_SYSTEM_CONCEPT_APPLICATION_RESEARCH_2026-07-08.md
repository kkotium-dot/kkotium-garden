# 디자인 시스템 고도화 전략 — 컨셉 트렌디 적용 (2026-07-08)

Authoring: DESKTOP(심층 리서치). 권위: 화이트→컨셉 전환 설계 근거. 방향: Claude Design 컨셉(Retro-Pop Garden) 미감을 현 앱 기능/구조에 트렌디 재해석 + 꽃틔움 브랜드(핑크+레드·손그림 로고·꼬띠·정원) 유지 + ADHD 집중. 꼬띠/로고 벡터 적용 예정.

================================================================
## 0. 핵심 원칙 한 줄
================================================================
**"감성은 가장자리에, 절제는 중심에"** — 매일 반복하는 데이터 작업 화면일수록 장식 제거, 어쩌다 만나는 전환 순간(빈상태·온보딩·성공·히어로)일수록 브랜드 감성 채움.

================================================================
## 1. 영역별 감성/절제 매핑
================================================================
| 영역 | 감성/절제 | 지침 |
|---|---|---|
| 온보딩 | 감성 최대 | 꼬띠·풀컬러 정원·연핑크 배경 허용·"play first" |
| 빈 상태 | 감성 강 | 꼬띠+스캘롭 카드+한 문장+단일 CTA·안내:재미=2:1 |
| 히어로/대시상단 | 감성 중 | 분수대 매출 히어로·단 KPI 4카드는 숫자/트렌드/스파크만(Stripe 절제) |
| 성공 축하 | 감성 강(짧게) | 꼬띠 표정+마이크로 컨페티·200~500ms·1회·해제가능 |
| 로딩 | 감성 약 | 정원 로더+정적 텍스트 병기 |
| 데이터 테이블 | 절제 최대 | 화이트·정렬 규칙·sticky 헤더·장식 금지 |
| 폼/입력 | 절제 강 | 위저드·명확 CTA·undo·마스코트 최소 |
| 밀집 리스트 | 절제 강 | 마스터 상태색만·일러스트 배제 |
근거: Carbonmade 리버스멀렛·Mailchimp Freddie·배민 셀프서비스 카드 절제·토스 인물그래픽.

## 2. 2026 트렌디 요소 — 4개만 취함
================================================================
- 부드러운 그림자(뉴브루탈 두꺼운 보더 대신 4px 레드 단색 그림자) · 유기적 형태(정원 곡선·bento) · 미묘한 그레인(히어로/카드만) · 표현적 헤드라인(디스플레이만 크게·본문 절제).
- 배제: 크롬/홀로그램·과한 안티디자인·혼돈형 뉴브루탈.
- 트렌드는 "전면 도입" 아닌 "선택적 악센트". 브랜드 코어는 장기자산·트렌드는 교체 가능 레이어로 분리.

================================================================
## 3. 꼬띠 마스코트 UX 가이드 (Clippy 반면교사)
================================================================
- **위치**: 온보딩·빈상태·성공·오류·도움말 툴팁. **비등장**: 데이터 테이블·폼·밀집·숫자 작업.
- **빈도**: 저빈도·세션당 1회·해제 후 해당 맥락 재등장 금지(Clippy 치명상=해제해도 복귀).
- **톤**: 차분 기본·baby schema(큰눈·둥근)·Duo식 죄책감 배제·토스식 조력자.
- **모션**: 200~500ms·reduced-motion 시 정적·차분에서 표정 숨김(확정됨).
- **필수**: 모든 등장에 닫기/숨기기(Mico 2025 기준).
- 규칙 코드화: "빈상태/축하/온보딩/오류만·세션1회·항상 해제·해제 시 재등장 금지·reduced-motion 정적" → 컴포넌트 강제로 Clippy화 원천 차단.

================================================================
## 4. 핑크 재도입 전략 (포인트→면·화이트 유지)
================================================================
단계(약→강): ①호버/선택 → ②카드헤더/섹션 악센트 → ③빈상태 배경 → ④대시보드 상단 존 → ⑤(최후·가역)배경 전면.
- 마켓컬리 원칙: 메인컬러 상단에·나머지 흰색·핵심(할인/구매버튼)만 강조.
- 각 단계 WCAG AA 전수 검증(연핑크+흰텍스트=실패 조합 주의)·ADHD 집중 저하 없을 때만 다음.
- 전환 동인=색 자체 아닌 "브랜드 일관성+주변 대비"(일관 브랜드색=인지도 +80%).

================================================================
## 5. 컴포넌트 트렌디 재해석
================================================================
- 카드: 스캘롭 유지+부드러운 레드 그림자(두꺼운 보더 대신)·데이터 카드 무장식.
- 버튼: 레드 CTA+부드러운 그림자·호버 마이크로 스케일/연핑크.
- 배지: 마스터 12색·색+아이콘/텍스트 병기(색맹).
- 빈상태: 꼬띠+정원·정보형/행동형/축하형 3종.
- 분수대 매출: 히어로 1개·물줄기=매출 은유+숫자/목표/스파크라인 필수 병기(감성+기능 화해).
- 네비: 정원 은유+기능 라벨 병기 필수(ADHD·Clippy 교훈).

================================================================
## 6. 장식·모션 3계층 (장식강도 토글 연동)
================================================================
- L1 기능 모션(항상): 상태 피드백 200~400ms·reduced-motion도 정적 유지.
- L2 브랜드 모션(기본 켜짐·차분서 축소): 컨페티·페이드·호버 스케일·reduced-motion 시 제거.
- L3 장식 모션(기본 꺼짐·화려서만): 꼬띠 표정·배경 미동·분수대 흐름.
- ADHD 안전: 자동재생·무한루프·5초+ 금지(또는 일시정지)·모션은 목적 있을 때만.

================================================================
## 7. 화이트→컨셉 전환 로드맵 + 재조정 신호
================================================================
- **Phase 0(현재)**: 화이트 중립·마스터 12색·3역할 폰트·장식토글 완성·꼬띠/로고 벡터 준비.
- **Phase 1(저위험)**: 빈상태·온보딩·성공에 꼬띠+정원·호버 연핑크. 검증=태스크 시간/오류율 불변.
- **Phase 2(컴포넌트)**: 카드 그림자·버튼/배지·분수대·네비. 검증=squint test(KPI/히어로 위계 지배).
- **Phase 3(핑크 확장)**: 카드헤더→대시 상단 연핑크. 검증=WCAG AA 전수·ADHD 집중.
- **Phase 4(배경·선택·가역)**: 배경 연핑크+그레인. 가역 토큰 롤백.
- **재조정 신호**: 과잉→태스크 시간↑·squint 실패·"산만" 피드백·reduced-motion 급증→롤백. 절제과잉→브랜드 인지 정체·"여느 어드민"·빈상태 이탈→감성 강화.
- **나침반**: "매일 반복 작업인가(절제) 어쩌다 만나는 순간인가(감성)?"

================================================================
## 8. 즉시 실행 권고 (6)
================================================================
1. Phase 1 착수: 꼬띠/로고 벡터를 빈상태·온보딩·성공 3곳만·호버 연핑크·데이터영역 무접촉.
2. 꼬띠 등장 규칙 코드화(위 §3).
3. 핑크 포인트→면 순서·단계별 AA 검증.
4. 분수대 히어로 승격+기능성 병기.
5. squint test 릴리스 체크리스트화.
6. 트렌드 4개만(그림자·유기형태·그레인·헤드라인).

================================================================
## 작업 유의사항/원칙 (신규)
================================================================
- **#228** 컨셉/브랜드 감성 적용은 "감성은 가장자리(빈상태·온보딩·성공·히어로)·절제는 중심(데이터·폼·리스트)". 화이트 유지하며 핑크는 포인트→면 단계 확장·각 단계 WCAG AA+ADHD 검증. 마스코트는 저빈도·해제가능·맥락한정(Clippy화 방지·코드 강제). 트렌드는 선택적 악센트 4개(부드러운 그림자·유기형태·그레인·표현적 헤드라인)·브랜드 코어와 레이어 분리. 판단 나침반="매일 반복(절제) vs 어쩌다(감성)". squint test로 위계 검증.
