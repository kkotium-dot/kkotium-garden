# 꽃틔움 가든: 네이버 스마트스토어 상품명 진단 엔진 & SEO 훅문구 개선 설계 리서치 (2026-06 기준)

> 출처: Desktop 심층 리서치 (2026-06-24). 권위 참조 문서 — 상품명 검사기/훅문구 엔진 개선 작업의 기준. 네이버 공식 정책과 업계 통념을 구분 표기.

## TL;DR
- **상품명 진단 엔진** = 네이버 공식 '표준상품명 9원칙'(50자 내외 권장·최대 100자, 중복단어 금지, 허용 특수문자 `( ) - · [ ] / & + , ~ .` 외 금지, 셀러명/이벤트성 문구 금지)을 규칙엔진으로 코드화 → 신호등(통과/주의/실패) + 그 위에 '경쟁강도(상품수÷검색량)' 기반 롱테일 키워드 **치환 제안**을 얹는 2단 구조. 단순 경고가 아니라 "이 키워드 빼고 이 세부키워드 넣으세요" 인라인 액션이 유료 툴 대비 차별점.
- **SEO 훅문구** = 노출 위치가 다른 두 곳(네이버 '이벤트 문구' 이벤트필드 EP / 상세페이지 상단 헤드라인)을 구분 타깃. 이벤트필드는 "2만원 이상 무료배송"식 구체 수치 혜택만 통과, "최선을 다하겠습니다"식 추상 홍보문구는 거부 → 생성기를 '혜택형(이벤트필드용)' + '감성형(상세페이지용 헤드라인+서브카피)'로 분리.
- **UI/UX** = 아이템스카우트·판다랭크식 '점수 + 신호등 + 인라인 제안' + 네이버 상품등록 '상품명 검색품질 체크' 모사 3색 체크리스트.

## 목표 1 — 상품명 실시간 진단 + 개선 제안 엔진

### (1) 네이버 공식 가이드 핵심 수치
| 항목 | 기준 | 출처 |
|---|---|---|
| 권장 글자수 | 50자 내외 권장·최대 100자(50자 이상 어뷰즈 확률↑) | 공식 |
| 실무 권장 | 25~30자(짧을수록 해당 키워드 노출 확률↑) | 통념 |
| 키워드 순서 | 브랜드/제조사→시리즈→모델명→유형(형태)→색상→소재→수량→사이즈→성별/나이→속성 | 공식 |
| 중복 단어 | 금지(동의어 네이버 자동처리, 반복 시 어뷰징 감점) | 공식 |
| 띄어쓰기 | 과도 붙임/분리 금지 | 공식 |
| 허용 특수문자 | `( ) - · [ ] / & + , ~ .` 만, 과다 금지 | 공식 |
| 금지 특수문자 | `! ? \ : * " < >`·이모티콘 → 감점/체크 실패 | 공식 |
| 셀러/쇼핑몰명 | 상품명 포함 금지 | 공식 |
| 언어/숫자 | 한글 기본·필요시 영문·아라비아 숫자, 그 외 금지 | 공식 |
| 이벤트/할인/쿠폰/적립 | 상품명 금지 → 별도 필드 | 공식 |

본질(9원칙 요약): "간결하게, 필요한 정보만."

### (2) 금지/제한어 (검색품질 체크 적발)
- 홍보성 수식어(저품질): 주문폭주·즉시할인·재입고·한정·가격·첫구매·문의·구매불가·공짜·품절·MD추천·**1위·선착순·임박·인기·가성비·최고**·상담·적립·땡처리·저렴·추천·신상품·공식·**정품**·이벤트·긴급모객.
- 과장/최상급('최고/최저가/1위') = 표시광고법 위반 소지 + 저품질.
- 전화번호 금지. 정치인명·유해물질 검출 모델명 자동 저품질.
- 예외: '명품'=모델명일 때만(쿠첸 명품철정), '고급'=고유상품명 포함시만. '정품'=상세페이지 안내 권장.
- 셀러 흔한 실수: 동일단어 반복·카테고리명 중복기입·브랜드 도용/타사 브랜드명·`+` 앞뒤 공백 미스.

### (3) 적합도 로직
- 네이버 랭킹 3축: **적합도·인기도·신뢰도**.
- 적합도 = "검색어가 상품명/카테고리/제조사·브랜드/속성·태그 중 어떤 필드와 연관도 높은지 + 카테고리 선호도" → **필드 연관도 + 카테고리 선호도** 2요소.
- 브랜드/제조사 **필드 분리** 권장: 검색어가 브랜드로 인식되면 상품명에 적은 것보다 '브랜드 필드' 매칭이 우선 노출 → 상품명에서 빼고 별도 필드 입력 유도.
- 카테고리-상품명 일치: 2025 플러스스토어 개편 후 중요도↑. 규칙: 타깃 키워드 1위 노출상품과 동일 카테고리.
- 상품명 SEO 스코어 = 강력한 감점(가점 아님). '실패' 판정은 실제 노출 손실 직결.

### (4) 개선 제안 로직 (툴 벤치마크)
- 핵심지표 **경쟁강도 = 상품수 ÷ 총검색수**(아이템스카우트·셀러마스터 공통, 낮을수록 좋음, 1 초과=공급과다).
- 롱테일 치환(디퓨저 예): 디퓨저 ~150만 → 실내디퓨저 ~10만 → 대용량디퓨저 ~2만. **메인키워드 유지**(빼면 적합도 손실)+세부키워드 조합으로 경쟁↓·구매의도↑.
- 세분화 축: 용도(실내/차량용)·규격(대용량/600ml)·타깃(30대/유아)·지역·시즌(입학선물).
- ROI 기준선(토스페이먼츠): 경쟁률 10:1~30:1 미만 연관키워드 = 판매 경쟁력 有. 이보다 높은 메인키워드 단독=주의, 구간 내 롱테일=추천.
- 구매의도 가중: 구체 검색어(유아 텀블러)=구매 결정단계, 검색량 낮아도 신규셀러 1P 진입 가능 롱테일 우선.

### (5) 신호등 체크리스트 (구현 판정 기준값)
| 진단 항목 | 통과🟢 | 주의🟡 | 실패🔴 |
|---|---|---|---|
| 글자수(공백제외) | 25~40자 | 41~50 또는 ~24 | 50 초과/극단 단문 |
| 메인키워드 포함 | 카테고리 매칭 1+ | — | 없음 |
| 세부키워드 수 | 2+ | 1 | 0 |
| 중복 단어 | 없음 | 동의어 1쌍 | 동일단어 2회+ |
| 특수문자 | 허용기호 소량 | 허용기호 과다 | 금지문자 1+ |
| 금지어/수식어 | 없음 | 경계 단어 | 홍보성 금지어 |
| 띄어쓰기 | 자연스러움 | — | 전체붙임/과도분리 |
| 카테고리 적합도 | 1위상품 동일 | 인접 | 불일치 |
| 브랜드 필드 분리 | 제외+필드입력 | 상품명 1회 | 반복/타사 브랜드 |
| 경쟁강도 | 10:1~30:1 롱테일 확보 | 1~3개 고경쟁 | 메인키워드만 |

> 네이버 자체 '상품명 검색품질 체크'([상품관리>상품등록>상품명] 하단)는 중복/금지문자/길이만 통과·오류 메시지 제공, 점수·제안 없음 → 꽃틔움 가든은 '점수+액션 제안'으로 확장하는 포지션.

## 목표 2 — SEO 훅문구

### (1) 노출 위치 구분
| 위치 | 노출 조건 | SEO 반영 | 글자수 |
|---|---|---|---|
| 이벤트 문구(이벤트필드 EP) | 쇼핑파트너센터 등록요청→검수 통과 | 혜택전달용·랭킹 가중 낮음 | 공식 미공개(추정) |
| 상품명 | 필수 | 핵심 SEO 필드 | 최대 100자(권장 25~30) |
| 상세페이지 상단 헤드라인 | 자유(에디터) | 직접색인X·체류/전환 간접기여 | 자유 |
| 검색결과 보조 | 자동 | — | 이벤트문구·배송태그·가격/할인·적립·리뷰 |

> 정정: "이벤트 문구 17/25/50자" 구체수치 = 공식 미확인(추정). 정확 maxlength는 쇼핑파트너센터 실측 필요. 스마트스토어 상품등록에 별도 '간략설명'·'AI 한줄카피' 전용필드 없음(공식 매뉴얼 미확인). '검색설정' Page Title/메타설명은 검색카드 아닌 외부 링크 공유 노출.

### (2) 이벤트필드 통과 규칙 (혜택형)
- ✅ 통과: "2만원 이상 구매 시 무료배송"·"신한카드 6개월 무이자"·"회원가입 시 500포인트".
- ❌ 거부: "무료배송/세일가" 단독, "끝까지 최선을 다하겠습니다"·"품질보증 무료A/S" 추상홍보, HTML태그·장식특수문자(★●◇) 반복, 단순 상품명/모델명. 제어문자(탭·엔터)·공백문자 금지.

### (3) 전환 카피 공식
- 상세 7단계(전환 2배 사례): 판매자정보→신뢰증거(인증)→문제제시→타깃 콕→핵심이득→후기→혜택(긴급성: 오늘만/선착순 사은품).
- 네이버에서 통하는 요소: 무료배송·당일발송·사은품·정량혜택 → 단 상품명 아닌 이벤트필드/상세페이지 배치(페널티 회피).

### (4) 차량용 디퓨저 예시
- 🔴 나쁨: "최고급 차량용 디퓨저 강력추천 1위 초특가"(금지어 다발·과장·저품질).
- 🟢 헤드라인: "운전석에 퍼지는 호텔 라운지 향" / 서브카피: "은은하게 2개월 지속, 흘림 없는 차량 전용 설계 · 오늘 주문 시 당일발송" / 이벤트필드(별도): "2개 구매 시 리필 1개 무료증정".
- 타깃 맞춤: 실용=가성비·대용량·지속력 / 프리미엄=향·디자인·선물.

### (5) 씨앗심기→상세 재활용
- 훅문구 = '헤드라인(15자 내외) + 서브카피(40~60자)' 2분할 생성 → 상세페이지 상단 메인카피로 이식.
- 생성기 이상형: 상품당 (a)이벤트필드용 혜택문구 1개 + (b)헤드라인+서브카피 3종(혜택강조·감성·신뢰) 동시 산출.

## 목표 3 — UI/UX 벤치마크
| 툴 | 특징 | 차용 |
|---|---|---|
| 아이템스카우트 | 키워드분석·검색최적화 탭·AI 상품명생성·카테고리선호도/경쟁강도 시각화 | 지표카드+차트, 검색최적화 탭 직접생성 |
| 판다랭크 | 판다AI 상품명/상세 10초생성·검색량/경쟁강도 한눈 | 원클릭 생성+즉시적용 |
| 셀러마스터 | 상품수÷월조회수 '비율' 한줄(낮을수록 좋음) | 단일 핵심지표 강조 |
| 스토어링크 | 부족요소 진단+챗봇 생성 | 진단→생성 연결 |
| 네이버 검색품질체크 | 통과/오류 메시지(점수·제안 없음) | 친숙한 통과/실패 멘탈모델 |

**깔끔·직관 패널 설계 베스트프랙티스**:
1. 상단 종합점수(0~100)+한줄코멘트 2. 신호등 체크리스트(3색·빨강 우선정렬) 3. 인라인 수정제안(문제단어 하이라이트→클릭시 빼고/넣기 1클릭+diff 미리보기) 4. 롱테일 추천패널(검색량/상품수/경쟁강도 3열+추천배지) 5. 실시간 글자수/바이트 카운터.

## 구현 로드맵 (단계)
- **1단계 MVP(규칙엔진)**: 9원칙 정규식·사전 기반 → 글자수/금지어/특수문자/중복/띄어쓰기 신호등. 금지어 사전=(2) 리스트 시드.
- **2단계 차별화(키워드 인텔리전스)**: 검색량·상품수(네이버 검색광고 키워드도구/자체 크롤링) 연동 → 경쟁강도 계산 → 고경쟁 메인키워드 10:1~30:1 롱테일 치환 제안(diff+1클릭).
- **3단계 훅문구 엔진**: '이벤트필드용 혜택형(수치필수·거부어 필터)' + '상세페이지용 헤드라인+서브카피' 분리. 카테고리/가격대/타깃 톤 분기.
- **4단계 UX 마감**: 종합점수+3색 체크리스트+인라인 제안 우측 고정 패널 통합.

## 전략 임계값
- 경쟁강도 ≥30:1 → '주의' 강등·롱테일 우선추천.
- 이벤트필드 거부율↑ → 거부어 사전 즉시 갱신.
- 네이버 검색품질 체크 규칙 개정시(과거 11/7 패치 전례) 규칙엔진 동기화.

## Caveats
- 이벤트필드 정확 글자수 = 네이버 비공개. 신호등 글자수 임계값은 상품명 공식수치+통념 결합 추정. 이벤트필드 maxlength는 쇼핑파트너센터 실측 필요.
- 표준상품명 가이드 = 법적 강제 아니나 신뢰도/SEO 스코어 페널티로 실제 노출 강작동(감점 회피용).
- 키워드 상품수/검색량(디퓨저 150만 등) = 시점변동 → 실시간/주기 갱신 필요.
