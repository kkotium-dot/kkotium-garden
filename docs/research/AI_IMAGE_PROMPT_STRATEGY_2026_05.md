## 0. 3대 핵심 결론

1. **AI 기본법 제31조 표시의무는 1인 셀러에게 직접 적용 안 됨.**
   과기정통부 가이드(2026.1.22): "AI를 도구로 활용해 자기 콘텐츠를 만드는 사람 = 이용자"
   → 셀러는 의무 대상 아님. 단 **꽃틔움 앱 자신은 인공지능사업자** → 사전고지+표시 의무.
   진짜 리스크는 표시광고법 제3조 = "AI 사용 여부"가 아니라 "실제 상품과 다른 오인 가능성".

2. **2026.5 실전 최적 = Firefly Image 5 (법적안전 허브) + Nano Banana Pro (합성/한글/라이프스타일) 듀얼.**
   대표님 외주 경험("Firefly보다 Nano Banana가 합성 자연스러움") 2026.5에도 유효.
   Firefly Pro $19.99/월 = 단일 허브에서 Firefly+Nano Banana Pro+GPT Image+Runway 무제한.

3. **꽃틔움 차별점 = 프롬프트 자산화 + 모델 적응형 라우팅 + 전환율 피드백 루프.**
   모델은 빠르게 교체돼도 "intent→model prompt 매핑 + CTR/CVR 누적 데이터"는 셀러 자산.

## 1. 법적 안전성 (한국, 2026.5 기준)

### 1-1. 표시광고법 제3조 — 진짜 리스크 축
- 금지: 거짓·과장 / 기만적 / 부당비교 / 비방 표시광고
- 위반 시: 매출 2% 과징금, 2년이하 징역 or 1.5억 벌금, 손해액 3배(개정안 5배 추진)
- 대법원 2019두36001: 직접표현뿐 아니라 "간접 암시·전체 인상"도 판단
- 핵심: AI 이미지가 **실제 상품·효능·후기와 다르게 소비자 오인** 시 위반
  → "AI라서 불법"이 아니라 "실물과 다르면 불법"

### 1-2. AI 기본법 (2026.1.22 시행) — 셀러 적용 제외
- 제31조: 인공지능사업자에 생성 사실 고지 + 결과물 표시(가시/비가시) + 딥페이크 표시 의무
- 위반: 시정명령 or 3천만원 이하 과태료
- **과기정통부 가이드: "도구로 활용하는 이용자는 의무 대상 아님"**
  (예: Sora로 영상 만드는 유튜버 = 이용자)
- 계도기간 최소 1년 (~2027.1.22 잠정). 실제 처분은 극히 예외적 경우만

### 1-3. 공정위 가상인물 표시 (2026.4.8 행정예고, 미확정)
- AI 가상인물이 추천·보증 주체로 등장 시 "가상인물 포함" 표시 의무
- **사람 없는 제품/배경 이미지는 적용 밖**
- AI 모델·인플루언서 컷 넣으면 라벨 의무 발생
- 2026.4.28 의견수렴 → 전원회의 의결 후 시행

### 1-4. 중국 도매 상세페이지 번역 — 이중 리스크
- 저작권: 도매업체 약관에 "사용권+가공 가부" 명시 확인 필수.
  1688 등 본토 이미지 무단 번역 = 한·중 저작권법 침해 가능성 높음
- 원산지표시: 대외무역관리규정 제76조 (한글/한자/영문 택1)
- 표시광고법: 실물과 색·치수·구성 다르면 거짓·과장

### 1-5. AI 홍보영상
- 셀러는 이용자(의무 없음). 단 효능 시각화·전후비교·가상전문가 영상은 표시광고법 위험 큼
- 안전: 실촬영 본체 + AI 배경/B-roll, "AI 영상" 자막

### 1-6. 네이버 대표이미지 정책 (2024.10.28)
- 등록불가: 상품명 불일치 / 가격정보 / 배송정보 / 홍보텍스트 / 저품질
- AI 명시적 금지 X → 실물과 동일 시각특성 + 금지요소 없으면 위반 아님
- 위반: 네이버쇼핑 미노출 + 클린프로그램

### 1-7. 합법성 체크리스트
| 항목 | 허용 | 위험 |
|---|---|---|
| AI 배경/씬 합성 | ✅ 상품 본체 시각특성 일치 시 | ❌ 색·재질 다르면 거짓·과장 |
| AI 제품 단독 누끼 생성 | ⚠️ 권장 X (실물 일치 보장 어려움) | ❌ 실물 차이 |
| AI 가상모델 후기/추천 | ❌ 가상인물 표시 + 거짓·과장 | "가상인물 포함" 표시 시 일부 |
| 중국 상세 원본 번역 | ⚠️ 약관+표시광고법 통과 시만 | ❌ 가공금지 시 저작권 |
| 효능 전후 비교 AI | ❌ 거의 위반 | 실측 입증 시 좁게 |
| 썸네일 가격·할인·"1위" 텍스트 | ❌ 네이버 제재 | — |

## 3. 전환율 프롬프트 전략 (리빙/홈데코/잡화)

### 3-1. 썸네일 CTR
- 1000px+, 정사각 정중앙, 여백 대칭, 텍스트·랭킹·할인율 최소(2024.10.28 위반시 제재)
- 라이프스타일 컷 > 화이트 누끼 (단 핵심 시각특성 식별 가능해야)
- 실측: CTR 2.36%→3.08%(+0.72%p) 썸네일 개선 사례

### 3-2. 전환율
- 국내 이커머스 평균 2.06%(빅인 2021) ~ 1.33%(편차 큼)
- 기대치↑(디자인·품질·활용) + 불안제거(옵션·교환환불·소셜프루프) 양축
- 실측: 배너 개선만으로 0.18%→0.57%(3배)

### 3-3. 전환율 프롬프트 공식
**[제품+재질+색상] + [환경/배경] + [조명] + [카메라/렌즈/앵글] + [스타일/분위기]**

키워드 셋:
- 조명: soft window light / golden hour / softbox / rim light / morning sunlight
- 렌즈·앵글: 35mm / 50mm prime / macro / overhead flat lay / eye-level / shallow DOF
- 분위기: hygge / Scandinavian / Japandi / cozy / editorial / clean minimalist / warm pastel
- 구도: rule of thirds / negative space on [side] for copy / centered

### 3-4. 모델별 프롬프트 작성 차이
- Firefly: 짧고 명시적 + UI 파라미터(Aesthetic, Style intensity) 분리
- Nano Banana Pro: 대화형 멀티턴 + 참조이미지 + "유지요소 명시"(Keep product shape exact)
- GPT Image 2: 의도·맥락·제약 서술, 텍스트는 따옴표
- FLUX: 구조화 다단 + negative prompt 적극

### 3-5. 컨셉별 차별화
| 컨셉 | 시각요소 | 모델 |
|---|---|---|
| 저관여·감성(디퓨저/캔들) | 라이프스타일씬, 자연광, 톤온톤, 여백 | NanoBananaPro·Firefly5 |
| 고관여·기능성(주방/청소) | 디테일컷, 사용장면, 매크로 | GPTImage2·Firefly5 |
| 가격高(프리미엄) | 미니멀·럭셔리, 풍부한 그림자 | Firefly5·NanoBananaPro |
| 가격低(대중잡화) | 컬러풀·가독성 | NanoBananaPro·FLUX |

## 4. 꽃틔움 앱 적용 (게이트3 광합성 샤워 개선)

### 4-1. 합법+전환율 워크플로우
```
1. 실촬영/공식실사 (상품 본체) ← 표시광고법 회피 기본
2. 누끼 (어도비 익스프레스 or Firefly Remove BG) ← 대표님: 익스프레스로 충분
3. 배경/씬 생성 (Nano Banana Pro, 멀티참조)
4. 톤·색보정 (Firefly Image 5 편집 or 익스프레스) ← 대표님: 파이어플라이 편집 나쁘지않음
5. 한글 헤더 (익스프레스/PS 별도 합성, 썸네일엔 X)
6. 검수 게이트: 실물일치 / 가상인물 / 어뷰징텍스트 / AI메타데이터
7. 등록후 30일 CTR/CVR 회수 → 프롬프트 점수 업데이트
```
**3대 원칙**: ①상품은 실촬영·배경만AI ②셀러=이용자 포지셔닝 ③꽃틔움=사업자(사전고지)

### 4-2. 프롬프트 자산화 스키마 (핵심)
저장 필드: id / intent_tag / product_category / model / model_version /
base_prompt(모델중립) / model_specific_prompt / negative_prompt /
parameters(seed,ratio,resolution,style_intensity) / reference_image_ids /
output_image_id / legal_flags(가상인물,텍스트,실물사용,SynthID,ContentCredentials) /
internal_score(1-5) / business_metrics(listing_id,impressions/clicks/ctr/cvr/revenue 30d) /
lineage(parent_prompt_id,branched_from) / governance(approved_by,deprecated,timestamps)

**결정적 가치**: business_metrics 자동 업데이트 → "어떤 의도 프롬프트가 우리 카테고리에 먹히는지" 정량 누적

### 4-3. 모델 적응형 어댑터
intent("lifestyle_kitchen_top_down") + inputs → 모델별 프롬프트 자동변환
- Firefly: 짧은서술 + UI파라미터
- NanoBananaPro: 멀티턴+참조+유지요소명시
- GPTImage2: 의도·제약 서술
- FLUX: 구조화+negative
→ 이 어댑터 레이어 = 꽃틔움 moat. 모델 교체돼도 매핑 라이브러리 누적가치
