> 작성: 2026-06-04 (Desktop 리서치) | 성격: 영구 운영 기준 + 의사결정 매뉴얼
> 용도: 모든 디자인/이미지 작업 전 항상 펴보는 기준. 도구의 능력·한계를 가정하지 않고 정확히 파악하기 위함.

## 0. 한 줄 요약 (가장 중요)

**우리 도구 중 "글자로 사진을 새로 만들어내는(text-to-image)" 도구는 단 하나도 자동으로 못 한다.**
- 신규 이미지 생성 = **Adobe Firefly 웹앱에서 대표님이 수동 1-click** (소비자 플랜은 API 없음).
- 그 외 모든 도구(Adobe MCP, Claude Design, Figma, Canva)는 **기존 이미지를 가공·조립하거나 코드·레이아웃을 생성**할 뿐.
- 따라서 "무드 배경 만들어줘" = 대표님 Firefly 수동 작업 → 결과물을 MCP로 가공, 이 흐름이 유일하게 작동.

**4단계 파이프라인 + 단계별 담당 도구:**
1. **생성(GENERATE)**: Adobe Firefly 웹 (Firefly Image Model 5 — 상업 안전·indemnity 보장)
2. **가공(PROCESS)**: Adobe "for creativity" MCP (누끼, 네이버 규격 크롭, 확장, 색보정)
3. **조립(ASSEMBLE)**: 상세페이지/배너 = Canva, 앱 UI = Figma + Claude Design
4. **발행(PUBLISH)**: 네이버 하드룰 준수하며 업로드

## 1. 도구별 능력·한계 (역할 분담)

### Adobe Firefly (생성 엔진 / 대표님 무료 사용 가능)
- **Adobe 자체 모델 (Firefly Image Model 5 등)**: 2025-10-28 출시 최신 플래그십. 네이티브 4MP 사진급. **유료 플랜에서 IP 면책(indemnity) 보장** — 출력물 저작권 분쟁 시 Adobe가 방어. 주요 AI 이미지 도구 중 유일.
- **파트너 모델 (FLUX, Gemini/Nano Banana, GPT Image, Imagen, Runway, Ideogram 등 15종)**: 상업 사용은 가능하나 **Adobe 면책 없음 = 법적 책임이 대표님에게**. 최종 판매용 이미지에는 부적합.
- **자동화**: Firefly Services API는 **엔터프라이즈 전용**(월 ~$1,000 최소 약정, 4~8주 영업 사이클). 소비자 플랜엔 API 없음 → **사실상 수동 웹 UI만**.
- **약점**: 한복 등 한국 전통 의상·문양을 부정확하게 렌더링(기모노화 등). 전통 모티프는 실사진 또는 전문 파인튜닝 권장 + 반드시 사람 검수.
- **가격(소비자)**: Standard $9.99/월(2,000 크레딧) / Pro $19.99 / Pro Plus $49.99 / Premium $199.99. 유료는 자체 모델 무제한.
- **★ 면책 적용 범위 한계(유료 자체 모델이라도)**: 출력물 "저작권"만 보호. 상표권·퍼블리시티권 X, 실존 브랜드/인물/캐릭터 명시 프롬프트 X, 사용자 업로드 참조이미지 X, 베타 기능 X, 생성 후 직접 합성물 X.

### Adobe Express (무료 사용 가능)
- 템플릿 디자인, 텍스트/사진 편집, 배경 제거, 브랜드킷, 배치/리사이즈, 애니메이션 + 에디터 내 Firefly 생성 기능.
- 콘텐츠/템플릿/스톡은 디자인 내 상업 사용 허용. 내장 Firefly 기능은 유료 플랜에서 면책 상속.
- **주의(이름 함정)**: "Adobe Express Developer MCP"는 애드온 개발자용 문서 도구지 디자인 자동화 커넥터가 아님. 디자인 자동화는 아래 "Adobe for creativity" MCP가 담당.

### Adobe "for creativity" MCP (가공 주력 / 약 50개 도구)
- **확정된 능력 경계 (Adobe FAQ + 실제 도구 목록 확인)**: 이미지 가공·편집 + 자산 관리. **text-to-image 생성 불가.** (Adobe FAQ가 "text to image / Gen Fill"은 별개 제품인 Firefly AI Assistant 소관이라 명시.)
- **주요 도구**: 배경제거, 크롭/리사이즈, 벡터화, 생성형 확장(아웃페인팅 — 입력 이미지 필요), 피사체 선택, 색/노출 보정, 스톡 검색·라이선스, Firefly Board 조립(기존 자산 URN만), 템플릿 도구, 비디오 리사이즈/컷.
- **현실 깊이 주의**: Photoshop이 아닌 **Express 등급**으로 동작. 단일 정밀 작업보다 무인 배치 가공(배경제거·포맷변환·리사이즈)에 강함. 느릴 수 있음(9:16 리프레임 ~3분 측정 사례).

### Adobe Experience Manager (AEM) MCP — **우리와 무관**
- 엔터프라이즈 웹 CMS/DAM 도구. 솔로 셀러 무관. 연결 해제 유지(도구 혼선 방지).

### Adobe Marketing Agent MCP — **우리와 무관**
- 엔터프라이즈 마케팅 오케스트레이션(캠페인/오디언스/저니). 솔로 셀러 무관. 연결 해제 유지.

### Figma (현재 Starter 플랜)
- **MCP 한도(공식)**: Starter = **읽기 도구 월 6회** (청구주기 리셋). **쓰기 도구(use_figma, create_new_file)는 무제한 면제.** Pro = 일 200회.
- ★ 핵심: **만들기/쓰기는 무제한, 되읽기(검증)만 월 6회.** 그러니 빌드는 마음껏, 읽기는 아껴서.
- **강점**: 디자인 시스템, 변수/토큰, 컴포넌트 라이브러리, 정밀 벡터/UI, 개발 핸드오프. 우리 앱의 디자인 시스템 본진.
- **한계**: 사진 편집·합성·이미지 생성 불가.
- **업그레이드 시점(Starter→Pro $16/월)**: 파일 3개 초과 OR 코드생성용 디자인 읽기가 월 6회 초과 시.

### Claude Design (Anthropic, 2026-04-17 출시 리서치 프리뷰)
- 프롬프트·문서·이미지·코드·Figma를 받아 **실제 프론트엔드 코드(HTML/CSS/JS, React)**, 프로토타입, 슬라이드, 랜딩페이지, 디자인 시스템 생성. Canva/PDF/PPTX/HTML/Claude Code로 내보내기.
- **강점**: 빠른 탐색("Figma로 1개 만들 시간에 10개 방향"), 코드형 출력 → Next.js 프론트로 바로 연결.
- **한계**: 토큰 인식이지 패턴 인식 아님(컴포넌트 변형·접근성·모션은 아직 약함), 프론트만(백엔드 X), 사진 생성 X. 첫 출력 ~70% 완성도, 2~3회 반복 필요.
- **Figma와의 관계**: Claude Design(탐색·프로토타입) → Figma(시스템화·정밀) → Canva(마케팅 발행). 보완 관계.

### Canva (MCP 커넥터)
- **MCP 능력**: 텍스트/스펙에서 디자인 생성, 편집, 라이브러리 검색, 자산 업로드, 폴더 관리, PDF/PNG/JPG/PPTX/MP4 내보내기. **리사이즈는 Pro+**, **데이터 오토필+브랜드템플릿은 엔터프라이즈 전용.**
- **라이선스**: Free/Pro 콘텐츠는 **완성 디자인 안에서** 상업 사용 OK. **단독 판매 금지.** AI 생성물은 별도 약관(상업 OK, 독점권 X, 면책은 엔터프라이즈 전용).
- **최적 용도**: 상세페이지 그래픽, 시즌 배너, SNS — 가장 빠름. 앱 UI는 Figma/Claude Design.

## 3. 절대 한계 & 흔한 착각 (반드시 고정)

1. **AI가 글자로 제품 사진을 못 만든다** — 생성은 Firefly 웹 수동, MCP는 text-to-image 불가.
2. **Adobe MCP는 합성(이미지 위 이미지)·text-to-image 불가** — 기존 이미지 가공/조립만.
3. **Figma는 사진 편집·이미지 생성 불가.**
4. **Figma Starter = MCP 읽기 월 6회** (쓰기 무제한). 자동화 설계 시 이 점 반영.
5. **파트너 모델(FLUX/Nano Banana/GPT Image/Imagen/Runway/Ideogram)은 Adobe 면책 없음** — 최종 판매물 금지.
6. **Firefly 무료 = 면책 없음, 베타 = 면책 없음, 상표/퍼블리시티/직접 업로드 참조이미지 = 면책 범위 밖.**
7. **Canva: 단독 판매 금지, AI 콘텐츠 면책은 엔터프라이즈만, Pro 템플릿은 Canva 링크로만 배포.**
8. **네이버 대표이미지: 텍스트·테두리·무관물 금지, 기본가 구매가능, 1:1(~1000×1000, 폭 ≥1,300px 권장)** — 위반 시 노출 제외 위험.
9. **Firefly Services API는 엔터프라이즈 전용**(월 ~$1k 최소, 4~8주 영업).
10. **AEM MCP·Adobe Marketing Agent MCP는 엔터프라이즈 도구** — 솔로 셀러 역할 없음.

## 4. 네이버 스마트스토어 이미지 하드룰 (도구 선택을 좌우)

**대표이미지(썸네일):** 최소 160×160 정사각, **권장 폭 ≥1,300px, 1:1 정사각**(1000×1000 권장), 최대 20MB, JPG/JPEG/GIF/PNG. **제품만, 깔끔하게.** 금지: 홍보문구(무료배송/1위 등), 테두리/외곽선/장식, 무관 제품, 기본가 미구매 상품 → 노출 제외 위험.

**상세페이지:** 권장 폭 **860px**, 세로 무제한이나 ~3,000px 초과 시 분할(섹션당 ~1,100px 편안). JPG, 품질 ≥90. **방문자 75~80%가 모바일** → 텍스트 1.5배, 폰 검수 필수.

## 7. 주의 (버전 민감)

AI 도구 약관·모델 목록·가격·MCP 한도는 자주 바뀜. Firefly 파트너 모델 목록(2026-02-27 문서), Firefly 가격(2026-02), Figma 가격/한도(2025-11-17 업데이트, 2026-03-18 AI 크레딧 시행), Canva CLA(2026-04 업데이트)는 **분기별 재확인**. Claude Design은 리서치 프리뷰(2026-04-17)라 변동 가능. 면책은 법률 자문 아님 — 고위험 사용은 변호사 상담. 한복 등 한국 전통은 AI 정확도 약함 — 실사진/전문 파인튜닝 + 사람 검수.
