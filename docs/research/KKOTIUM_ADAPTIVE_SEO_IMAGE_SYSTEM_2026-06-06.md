## 0. TL;DR (핵심 3줄)

1. **상품 품질 점수(0~100)로 3모드 자동 분기**: 간편(70↑, 상세페이지 크롭+최소 SEO) / 보강(40~70, 문구·이미지 보강) / 주력신규(40↓ 또는 정책위반, B안 풀 파이프라인). 분류기 = OpenCV 정량지표 1차 + VLM 2차(경계 케이스만).
2. **네이버 SEO ROI 우선순위**: 카테고리 → 속성 전체 → 상품명(50자) → 태그 10개 → 추가이미지. 텍스트 SEO는 전 상품 무료 일괄, 이미지는 모드 차등 투자.
3. **씨앗심기(저관여·실용) vs 온실아틀리에(고관여·감성)**: 키워드·톤·이미지 강도를 라인별 차별화. 한 스토어에서 brand_line 필드로 구분.

## 1. 디자인 강도 자동 분류기 — "정량 1차 + VLM 2차"

### 네이버 랭킹 맥락
- 공식: 적합도·인기도·신뢰도·개인선호도 4요소만 공개. "35/40/25"는 비공식 추정. 인기도는 카테고리별 상이.
- 이미지는 적합도+신뢰도 양쪽 반영 → 저품질 이미지 = 신뢰도 감점 + 클릭률 저하로 인기도까지 깎는 이중 손실.

### 1차: 정량 게이트 (OpenCV/sharp, 무료·수십ms)
| 지표 | 측정 | 배점 |
|---|---|---|
| 해상도 | sharp metadata, 긴 변 1000px↑ | 20 |
| 선명도 | 라플라시안 분산 cv2.Laplacian.var() | 20 |
| 배경 단색도 | 가장자리 픽셀 분산/지배색 비율 | 15 |
| 정사각 적합 | 1:1 크롭 가능 영역 | 10 |
| 워터마크/텍스트 | 엣지·OCR 휴리스틱 | 15 |
| 상품 비중 | 누끼/엣지 피사체 점유율 | 20 |

### 2차: VLM 판정 (경계 40~70 또는 텍스트 의심만)
- GPT-4o(mini) vision, "대표이미지 적합성 평가 전문가" 역할 + 0~100 점수 + 사유 JSON 강제.
- detail:'low' = 85토큰 고정, 1만 장 ~$2 수준. 정량 게이트로 명백한 고/저 먼저 걸러 VLM 호출 20~30%로 제한.
- ★ VLM은 텍스트·워터마크 과소평가 → OCR/엣지 검사 반드시 병행.

### 모드 분기 (운영하며 보정)
- **모드1 간편(70↑)**: 상세페이지 1:1 크롭 + 최소 SEO(상품명·카테고리·속성·태그).
- **모드2 보강(40~70)**: 추가 문구/배너 생성, 톤 보정, 추가이미지 슬롯.
- **모드3 주력신규(40↓/정책위반/저작권위험)**: B안 풀 파이프라인(누끼+AI배경+합성).

### 위탁배송 특수 판단
- 도매처 이미지 = 다수 셀러 공용 → 네이버 카탈로그(가격비교) 묶임 + 중복 페널티. 주력은 모드3로 고유 이미지화해 묶임 회피.
- 도매처 가공·사용 허용 범위 상품별 확인 선행(분쟁 리스크).

## 2. 네이버 SEO 전범위 체크리스트 + ROI 우선순위

| 순위 | 항목 | 작업 | ROI 근거 |
|---|---|---|---|
| 1 | 카테고리 | 최하위까지 정확 매칭 | 적합도 핵심, 틀리면 미노출 |
| 2 | 속성 | 전 항목 채우기 | 봇 크롤링으로 미기재 키워드도 노출 |
| 3 | 상품명 | 50자, 메인+롱테일, 중복·수식어·특수문자 금지 | 적합도+신뢰도, 검색품질 체크 통과 |
| 4 | 태그 | 태그사전 등록 10개 | 동의어·용도·상황 보강 |
| 5 | 대표이미지 | 1:1, 1000px+, 단색배경, 텍스트 금지 | 클릭률→인기도, 위반 시 제재 |
| 6 | 추가이미지/동영상 | 정보·연출 보강 | 체류·전환 상승 |
| 7 | 상세페이지 | 정보+설득, 모바일 가독성, 가로 860px+ | 체류·전환, 봇 키워드 크롤링 |

**1인 셀러 전략**: 1~4(텍스트 SEO) 전 상품 무료 일괄 자동화 먼저, 5~7(이미지) 모드 차등. 초보는 대표키워드 대신 롱테일(세부) 키워드 우선.

**리뷰 중요도**: 소비자 97.2% 구매 전 후기 확인, 72.4%는 후기 없으면 미구매(한국소비자연맹 2021). → 동영상·사진 리뷰 유도가 전환 직결.

## 3. 씨앗심기 vs 온실아틀리에 차별화 매트릭스

| 축 | 씨앗심기(저관여·실용) | 온실아틀리에(고관여·감성) |
|---|---|---|
| 검색의도 | 즉시·습관구매 | 탐색·비교·후기 |
| 키워드 | 실용·가격·용도 단순조합 | 감성·후기·비교·선물 롱테일 |
| 상품명 톤 | 기능·스펙 직설 | 분위기+기능 균형 |
| 이미지 강도 | 모드1·2 중심(간결 정보형) | 모드3 중심(연출·합성) |
| 상세페이지 | 짧고 핵심, 스펙·배송 | 길게, 비포애프터·후기·스토리 |
| 톤앤매너 | 밝고 명료(레드/크림 절제) | 무게감·프리미엄(연출·여백) |

한 스토어: 카테고리·태그로 라인 구분, brand_line 필드로 SEO 템플릿·이미지 강도 분기. 디자인 시스템(레드 #E62310, 크림 #FAF8F3, Pretendard, 꼬띠)은 씨앗심기=마스코트·포인트 적극, 온실아틀리에=절제·프리미엄.

## 4. 상세페이지→썸네일 크롭 절차 + 정책 가드

1. 긴 상세 이미지에서 후보 영역 → smartcrop-sharp 주제인식 1:1 크롭(또는 sharp strategy.attention/entropy). 피사체 5~8% 여유 패딩.
2. 1000px+ 리사이즈, JPG/PNG, 20MB 이하 규격화.
3. **정책 가드(자동)**: OCR로 텍스트/워터마크 검출 시 → 텍스트 없는 영역 재크롭 또는 모드3 누끼 전환. 가격·배송·홍보 문구, 외곽선/도형/마크, 무관 상품 검사(2024.10.28 위반 시 제재).
4. 단색 배경 아니거나 피사체 비중 낮으면 누끼+단색 합성으로 규격화(B안 재사용).

## 5. 앱 통합 설계 (Next.js / Prisma / Supabase / Vercel)

### 데이터 모델 확장
- Product에 brand_line(SEED|GREENHOUSE), quality_score(Int), recommended_mode(SIMPLE|ENHANCE|NEW), quality_reasons(Json).
- asset_jobs job_type 확장: QUALITY_ASSESS → THUMB_CROP / SEO_TEXT / SEO_IMAGE / BG_SWAP(B안). 모드별 job 체인 다르게 큐잉.
- SeoFields 테이블: 상품명/카테고리/속성/태그/alt 구조화 저장, 라인별 템플릿 이력.

### 워크플로우 UX
1. 상품 투입(도매매 URL/이미지) → QUALITY_ASSESS 자동 실행 → 점수+추천모드+사유 표시.
2. 관제탑 매트릭스에 모드 컬러 배지. 셀러 1클릭 승인/변경.
3. 모드별 job 체인 자동 큐잉: 간편=크롭+SEO텍스트 / 보강=+이미지보강 / 신규=+B안.
4. 일괄 처리: 동일 모드 다수 배치 승인, 예외만 사람 개입.

### 운영 효율
- 정량 게이트로 VLM 비용 절감, 텍스트 SEO 전상품 자동 일괄, 이미지 모드 차등.
- 최소 클릭(추천 승인형), 배치 처리, 사람 개입 최소.
- 기존 B안 파이프라인을 모드3 job으로 재사용.

## 6. 구현 로드맵

**1단계(~2주)**: 정량 분류기(sharp+라플라시안/해상도/배경/피사체) + 모드 추천 + 관제탑 배지. 텍스트 SEO 자동 일괄(카테고리·속성·태그·상품명 검색품질 체크). 비용 0, 최速 ROI.
**2단계(~3~4주)**: smartcrop 썸네일 크롭 + 정책 가드(OCR) + 추가이미지 슬롯. asset_jobs에 THUMB_CROP/SEO_IMAGE.
**3단계**: VLM 2차 판정(경계만, detail:low), 라인별 SEO·연출 템플릿 분기, B안 모드3 연결.

**벤치마크/임계값**: VLM 호출 ≤30% 유지 / 모드 추천 수용률 ≥70%면 분류기 신뢰. 모드3 과다 시 임계값 하향(과잉 방지). 카탈로그 묶임률 모니터링. 상품명 검색품질 빨간불 0건.

## 7. 작업 유의사항 (영구 + 본 트랙)

- **3모드 원칙**: 모든 상품 동일 작업 금지. 품질 점수로 간편/보강/신규 분기. 과잉 작업 방지가 1인 셀러 ROI 핵심.
- **간편 모드 우선**: 상세페이지 양호하면 크롭+최소 SEO로 끝. 명화 디퓨저가 대표 사례(상세페이지 양호 → 간편 모드).
- **SEO 전범위**: 텍스트(카테고리·속성·상품명·태그) 전상품 일괄 + 이미지 모드 차등. 씨앗심기/온실아틀리에 라인 분기.
- **대표이미지 = 흰배경 누끼/단색, 텍스트 금지**(네이버 2024.10.28). 크롭본 OCR 가드 필수.
- **위탁 이미지 카탈로그 묶임 주의**: 주력은 고유 이미지화. 도매처 가공 허용 확인.
- 기존 영구원칙 계승: B안 제품교체 / 이미지 생성 최고품질 모델 / 브라우저 반자동 / 비가역 0(#46) / 각 프로그램 적재적소.

## 8. 한계 / 재확인

- 네이버 랭킹 가중치·이미지 SEO 산식 비공개("35/40/25" 추정치). 인기도 카테고리별 상이.
- 권장 픽셀 2차 출처 상충(1300px vs 1000×1000). 1:1·20MB·JPG/PNG·최소 160×160은 교차확인. 현행 권장은 스마트스토어센터 원문 재확인.
- VLM은 텍스트·워터마크 과소평가 → OCR/엣지 병행 필수. 오픈소스 미학 예측기 편향 보고 → 보조 지표만.
- 도매처 이미지 저작권·가공 범위 공급사별 상이, 상품별 확인 선행.
