# 해외 직소싱 영구 Baseline — 셀렉트 편집샵 큐레이션 셀러 단계별 (2026.05)

**문서 작성일**: 2026-05-08
**기준 시점**: 알리바바 Accio Agent 1,000만 MAU 도달 + 네이버 AI 쇼핑 에이전트 베타 + 한국 KC 직구 금지안 백지화 후
**대상**: 꽃틔움 가든 앱 사용자 (1인 디자이너 출신 스마트스토어 새싹셀러)
**용도**: 새싹→파워→빅파워 성장 로드맵의 *해외 직소싱 도입 시점·방법* 영구 baseline
**다음 갱신 권고**: 2026.11 (반기)

## TL;DR (3줄 결론)

- **지금(매출 200만 이하)은 해외 직소싱을 "시작"하지 말고 "준비"만 하라.** 도매매 운영을 안정화해 월매출 600만 원을 먼저 찍는 게 압도적으로 ROI가 높다. 그 사이 해야 할 단 두 가지: (1) 1688·SUPER DELIVERY 회원가입 + 사업자통관고유부호 발급(기술적 준비), (2) 큐레이션 콘셉트의 스타일 가이드 1.0 문서화(브랜드 준비).
- **파워 등급(월 800만~3,000만)에 도달한 순간이 결정적 분기점이다.** 디자이너 강점을 살리려면 *중국 1688 직접구매(슈퍼레이블/굿게이트 등)* + *일본 SUPER DELIVERY*의 2채널을 먼저 열어야 한다. Faire/Ankorstore는 한국 셀러가 *바이어*로 입점하기는 사실상 불가능하므로 **자체 브랜드 보유 후 *셀러*로 역진출**하는 게 현실적인 활용 경로다.
- **2026년의 진짜 변화는 두 가지뿐이다**: (1) 알리바바 Accio Agent(B2B 소싱 AI, 2026.3 기준 월간 1,000만 명)가 *공급사 발견·RFQ 자동화*의 진입장벽을 무너뜨렸고, (2) 네이버 쇼핑 AI 에이전트(2026.2 베타)가 *상품 정보 요약·비교*를 자동화하면서 일반 위탁 셀러의 차별화 여지를 좁혔다. 이 두 변화는 **"누구나 같은 상품 소싱"의 진입장벽을 낮추되, "셀렉트 안목·자체 브랜딩·라이프스타일 제안"의 가치는 오히려 올렸다.** 디자이너 출신 셀렉트샵 셀러에게는 명백히 호재다.

## Key Findings (요약 결론 8가지)

1. **해외 직소싱의 진입 시점은 "월매출 600만 원 + 누적 판매 300건"이다.** 이는 스마트스토어 파워 등급(최근 3개월 누적 판매 300건+/800만 원+) 직전이며, 이 지점에서야 *샘플 비용·MOQ·통관비 회수*가 산수상 성립한다. 그 이전엔 **도매매 + 1688 구매대행(슈퍼레이블·도매꾹 1688 상품관 등)** 조합이 사실상 유일한 합리적 선택.

2. **디자이너 강점이 실질적 차별화로 작동하는 카테고리는 4개로 좁혀진다**: 일본 디자인 문구·생활용품(SUPER DELIVERY), 유럽 디자이너 도자기·인테리어 소품(Faire/Ankorstore 출신 브랜드의 한국 독점), 동남아 핸드메이드 패션·잡화(베트남 호치민·태국 Pratunam·발리), 그리고 **중국 1688 OEM 무지+자체 브랜딩**. 단순 1688 가격 사입은 디자이너 강점이 0에 수렴한다.

3. **2026년 5월 시점, 한국 셀러의 해외 직소싱 환경은 "역대 최고로 우호적"이다.** 윤석열 정부의 KC 인증 직구 금지안(2024.5 발표)은 정부 교체로 사실상 백지화되어 *목록통관 USD 150 / 미국 USD 200* 한도가 그대로 유지되며, 엔화는 100엔당 860원대 고착(2026.1Q 기준), 알리바바·테무의 한국 직진출은 *위탁 셀러*는 압박하지만 *큐레이션 셀러*는 오히려 반사이익을 본다.

4. **AI 도구 5대장은 이미 셀러 표준 스택이다.** (1) Accio Agent (알리바바, B2B 소싱·RFQ·협상 자동화), (2) 1688 자체 이미지 검색 (상품 동일/유사 검색), (3) Wise Business (다중 통화 결제·외화 수취), (4) 드랩아트/캐럿AI/Instant Studio (상품 사진 라이프스타일 합성, 한 장 100~2,000원), (5) GPT-5/Claude 4 등 LLM (중국어→한국어 계약·스펙 번역, HS Code 1차 분류). 이 중 *진정한 진입장벽 파괴자*는 Accio와 라이프스타일 사진 AI 두 가지뿐이며, **나머지는 효율화 도구**다.

5. **법적 핵심 전환점은 "구매대행"에서 "사업자 직수입"으로 넘어가는 순간이다.** 직수입 = 사업자통관고유부호 발급(유니패스, 약 3시간) + 정식 수입신고 + 관세·부가세 *전체 금액 과세*(150달러 면세 不적용) + KC/식약처 인증 책임 *판매자에게 귀속*. 한·중 FTA는 700달러 이하 원산지증명서 제출 면제(단 발급은 보유)로 의류 등 4~13% 관세 절감 가능.

6. **Faire·Ankorstore는 "사용자가 *바이어*로 들어가는 채널"이 아니라 "사용자가 *판매자*로 진출하는 채널"이다.** 이들은 EU·미국 *현지* 독립 리테일러를 검증해 가입시키는 구조이며, 한국 사업자가 바이어로 가입해 미국·EU 디자이너 상품을 원가에 사오는 건 사실상 불가(배송지 검증·결제 카드 검증·세금 처리 모두 현지 기준). 빅파워+ 도달 후 *자체 브랜드*를 만들어 Faire에 *Brand*로 입점하는 게 현실 경로.

7. **꽃틔움 가든 앱의 해외 어댑터 우선순위는 명확하다**: 1688 OpenAPI(공식) > SUPER DELIVERY(웹 자동화) > Accio API(베타 모니터링) > Faire API(셀러로 역진출 시). 1688은 OFF·알구고·굿게이트 등 한국 SaaS 다수가 이미 공식 API 연동 사례를 검증했으므로, B2B 어댑터 패턴 PoC의 두 번째 어댑터로 1688을 채택하는 게 가장 합리적이다.

8. **1인 셀러가 망하는 패턴 7가지는 모두 "재고 동결"로 수렴한다.** SNS 트렌드 추격 사입 → 유행 끝물 도착, OEM 무지+한글 라벨 비용 누락, KC 인증 미인증 적발(과태료+계정 정지), 환율 헷지 미실시, 샘플과 양산품 차이, 가품 침해(상표·디자인), 계절 상품 배 운임 지연. **공통 처방: "샘플 1개 → 소량 30개 → 본격 200개" 3단계 검증을 절대 건너뛰지 말 것.**

### 2. AI 도입 변화 (2024~2026 최신 트렌드)

#### 2-A. 알리바바 Accio Agent — *진짜 게임 체인저*

**Accio의 진화 타임라인 (사실 확인 완료)**
- **2024.11**: Alibaba.com이 AI 기반 B2B 검색엔진 Accio 출시 (베타).
- **2025.8**: Accio Agent (Agent Mode) 정식 출시 — 10억 개 상품·5,000만 공급사 데이터 학습. 기존 "수 주" 걸리는 소싱 워크플로우를 "수 분"으로 압축. 70% 수동 작업 자동화. 출시 시점 200만 사용자 돌파.
- **2026.3**: Accio Work 출시 — 기업용 멀티 에이전트(분석·창작·물류 전문가). 월간 액티브 사용자 1,000만 명 돌파(MIT Technology Review, 2026.4 기준 — 알리바바 사용자 5명 중 1명이 AI 소싱 활용).

**Accio가 셀러에게 실제로 해주는 일**
1. *제품 컨셉 → 시장 분석 → 디자인 가이드 → 공급사 검증 → 일괄 RFQ → 협상 → 발주*까지 단일 대화로 완결.
2. 이미지·텍스트·음성으로 공급사 검색 (인스타에서 본 일본 도자기 사진 한 장 → 1688/Taobao/AliExpress 동일/유사 공급사 자동 매칭).
3. Hot-Selling Product Redesign 기능 — 잘 팔리는 상품의 변형 디자인 자동 제안.

**Accio의 한계 (정직한 평가)**
- **알리바바 생태계 잠금**: Alibaba.com·1688·Taobao 외 공급사는 검색 불가. 일본·유럽 디자이너 상품은 *직접 발견 채널이 아니다*.
- **마케팅·SNS 영역은 약함** (MIT Technology Review 인터뷰 — Mike McClary가 *제품 발굴은 압도적이지만 광고·SNS 질문은 여전히 부족하다*고 평가).
- **추천이 일반적(generic)일 때가 있어** 사용자가 도전하고 검증해야 한다는 일반적 한계.

**Accio 활용 시점**: **파워 등급 도달 후 1688 직접구매 전환 시점**. 그 이전엔 도매매 사용 중이라 ROI 미달.

#### 2-B. AI 번역 정확도 (중국어 → 한국어, 2026년 시점)

- GPT-5 / Claude 4 / Gemini 2.5: 제품 스펙·계약서 번역 정확도 약 95%+ (단, *전문 무역 용어*는 사람 검증 필수).
- 슈퍼레이블·OFF 같은 한국 SaaS는 1688 상품 페이지 통째로 한국어 자동 번역 + UI 통합 제공.
- **계약 자동화 패턴 (셀러 보고)**: 공급사 위챗 대화 → DeepSeek/ChatGPT로 요약·요지 추출 → NDA 초안 자동 생성 → 사람이 검토·수정.

#### 2-C. AI 이미지 생성 — 상품 사진 보강 워크플로우 (디자이너 강점 ×2)

**2026년 시점 한국 셀러가 실제 쓰는 도구 (가격대 100~2,000원/장)**
- 드랩아트(draph.ai): 상품 누끼 + 가상 모델 합성, 한국어 UI, 묶음 상품(최대 80개) 자동 배치.
- 캐럿AI(carat.im): 루마포톤·플럭스·리크래프트 다중 모델, 썸네일·상세페이지·인스타용 템플릿.
- Instant Studio (Shopify, 2026.3 출시): AI 아바타 재사용, 시즌 배경 자동 변경.
- Nano Banana Pro API (Google, 2026): 4그리드 내러티브 이미지 생성, 글로벌 셀러용.
- AMU Gen Studio: studio-product-commercial-photography 프롬프트 템플릿.
- 가비아 제디터(클릭엔): AI 상품 상세페이지 자동 생성 + AI 이미지 서비스(2026.4.15 출시).
- Photoroom·removebg: 배경 제거 무료.

**핵심 변화**: 상세페이지 한 장 만드는 데 *2주 + 수십만 원*에서 *반나절 + 만 원*으로. **디자이너 출신은 *프롬프트로 톤·매너를 일관되게 유지*할 수 있어 비디자이너 셀러 대비 결과물 품질 격차가 명확하다.**

#### 2-D. AI 가격 예측 + 환율 예측

- AliPrice 이미지 검색 + 가격 추적: Amazon·Ozon·Coupang·Shein 동일 상품 가격 비교, 가격 추이 그래프.
- Wise / 신한 Wise 송금: 자동 환전 (목표 환율 도달 시 자동 실행).
- 국가별 한국 셀러 보고: 환율 헷지보다 *실수요 대비 30% 선구매 + 70% 시장 환율 노출* 패턴이 가장 일반적.

#### 2-E. AI 통관 코드(HS Code) 자동 분류

- 굿게이트 ERP가 "상품등록 시 HS Code AI 자동 생성" 기능 신규 탑재(2026.1 기준).
- LLM에 상품 정보(소재·용도·치수·기능) 입력 → HS Code 6자리 후보 3개 + 관세율 자동 출력 → 관세사 최종 확인 워크플로우.
- **주의**: HS Code 1자리 차이로 관세 0% ↔ 13% 차이가 날 수 있어 **고가·대량 상품은 반드시 관세사 사전 자문(보통 5,000~30,000원/건)**.

#### 2-F. 진입장벽이 *낮아진* 영역 vs *오히려 차별화가 어려워진* 영역 (정직한 분석)

| 영역 | AI로 진입장벽 ↓ | AI로 차별화 ↑ |
|---|---|---|
| 1688 가격 사입 + 위탁 판매 | 장벽 거의 사라짐 (Accio·알리프라이스) — 누구나 같은 상품 발견 | 차별화 사실상 불가 |
| 상품 사진·상세페이지 제작 | 누구나 5만 원에 스튜디오급 결과물 | 톤·일관성·세계관은 디자이너만 유지 가능 |
| 중국어 협상 | 누구나 LLM으로 가능 | *관계 자본*은 AI 대체 불가 (장기 거래 단가·우선 출고) |
| 트렌드 분석·키워드 발굴 | Accio·네이버 데이터랩 통합 | 모두가 같은 트렌드를 보게 됨 |
| **셀렉트 큐레이션 안목** | AI로도 모방 어려움 | **디자이너 강점 그대로 유효** |
| **자체 브랜딩·세계관** | AI는 보조 도구일 뿐 | **디자이너 강점 +50%** |
| **라이프스타일 제안·스토리텔링** | 텍스트 생성은 가능, 일관성은 못 함 | **디자이너 강점 +50%** |

**결론**: *상품 자체*의 차별화는 AI로 더 어려워졌지만, *큐레이션·브랜딩·세계관*의 차별화는 AI 시대에 오히려 가치가 올라갔다. 디자이너 출신 정체성은 정확히 후자의 영역에 자산을 가진다.

### 3. 셀렉트 편집샵 큐레이션 관점 — 디자이너 강점 활용

#### 3-A. 일관성 있는 큐레이션 셀렉 패턴 (Style Guide 1.0 권고 양식)

**즉시 작성해야 할 큐레이션 헌법 — 4개 축**
1. **컬러 팔레트** (3+1 룰): 메인 2색 + 서브 1색 + 포인트 1색. 예: 베이지·아이보리·웜 그레이 + 머스타드. 모든 신규 상품은 이 팔레트에 들어와야 한다.
2. **소재 가이드**: 우드·세라믹·린넨·유리·라탄 중 4개 이내. *플라스틱·실리콘·메탈은 셀렉 제외*(예시) 등 제외 리스트도 함께.
3. **가격대**: 객단가 25,000~80,000원 / 평균 45,000원. 80,000원 이상은 *시즌 한정 5% 미만*.
4. **세계관 키워드**: "조용한 아침", "느린 식탁", "혼자의 의식" 등 3개 키워드. 모든 상세페이지·인스타 카피가 이 키워드와 정렬.

#### 3-B. 발굴 가치 있는 상품 카테고리 (디자이너 강점 매트릭스)

#### 3-C. 한국 셀렉트샵 워크플로우 (벤치마킹 대상)

- **29CM**: "감도 깊은 취향" 슬로건. 입점 브랜드 큐레이션 매우 까다롭고, 자체 콘텐츠(Editor's Pick, Inspiration) 비중 높음. 통신판매중개자 모델로 재고 부담 없음. **참고할 점: 콘텐츠가 곧 큐레이션이라는 인식.**
- **텐바이텐 (10x10)**: 백패커 운영, 문구·디자인 소품 강세. 개별 판매자 입점 모델. **참고할 점: 카테고리 깊이(문구만 수천 개 SKU).**

4. **'개미일기' 군소매체 기자(바이라인네트워크)** — 2주 10개 상품 109,000원 매출 사례 — *역설적 시사점: 단순 상품 등록만으론 매출 0원이 기본값. 큐레이션·브랜딩이 곧 매출이다.*
5. **i-boss 칼럼니스트 익명** — 4달 만에 빅파워 달성, *키워드+제품 선정에 70% 비중*. *실용 시사점: "내 수준에 딱 맞는 상품"이 1단계.*
6. **VPLATE 1인 셀러 가이드** — Copilot(GPT-4 Turbo)·Canva·removebg·뤼튼 무료 스택으로 1인 상세페이지 제작 사이클 1일 → 2시간 단축.
7. **슈퍼레이블 사용자 후기 다수** — 구매대행에서 1688 직접구매 전환 후 원가 5~10% 절감 + WorldFirst USD 충전으로 환전 수수료 1/6 절감.

#### 4-B. 면세 한도 (2026년 5월 시점 기준 변동 없음)

- **목록통관**: USD 150 이하 (미국발 USD 200 이하). 의약품·건강기능식품·식품·기능성 화장품 등 *목록통관 배제 대상*은 제외.
- **합산과세**: 동일 발송인 + 동일 입항일 또는 동일 B/L 분할 시 합산. 2022.11 동일 입항일 합산 기준은 삭제됨.
- **150달러 초과**: 전체 금액에 관세·부가세 부과(초과분만 X). 미국발은 200달러 초과 시 동일.
- **사업자 사입은 자동으로 일반 수입신고** — 면세 한도 무관하게 전 금액 과세.

#### 4-C. HS Code, 관세율, FTA 활용

- **한·중 FTA**: 700달러 이하 수입 시 원산지증명서 *제출 의무 면제*(단, 발급은 보유 필수). 수출자가 대한상의/세관에서 발급(1~2일, 비회원 7,000원).
- **활용률**: 한·중 FTA 수입 활용률은 80~90%로 매우 높음. 의류 4~13% 관세 절감.
- **사후적용**: 수입 후 1년 이내 협정관세 사후적용 신청 + 경정청구 가능.
- **AI HS Code 분류**: 굿게이트 등 ERP가 자동 생성, 단 *고가·대량은 관세사 검증 필수*.

| 수단 | 송금 수수료 | 환전 수수료 (마진) | 실수령액(1,000 USD 송금 시) | 적합 용도 |
|---|---|---|---|---|
| Wise (개인/Business) | 약 0.4~1% (고정+변동) | **0% (미드마켓)** | 약 990~995 USD 상당 | 1인 셀러 표준 |

**3가지 가능 패턴 (현실성 순)**:

### 8. 2026년 5월 시점의 결정적 변화

#### 8-A. 미·중 무역 분쟁 영향
- 2025.5 미·중 관세 일부 완화 합의 → 2026년 4월 현재 *기술 통제는 유지·소비재 관세는 완화*. 일반 라이프스타일 소비재는 한국 셀러에게 유의미한 변화 없음.
- 단, 2025년 트럼프 행정부 미국의 中 수입품 관세 인상 → 중국 제조사들이 *동남아·아프리카·라틴 아메리카·한국으로 수출 다변화* → **한국 셀러 입장에서 1688 가격 경쟁력은 오히려 개선**.

#### 8-B. 한·중 FTA 활용 확대
- 활용률 2024 80% → 2026 90%대(추정). 의류·가방·소품 4~13% 관세 절감.
- 2015.12.20 발효, 2026 = 11년차로 점진적 관세 철폐 진행 중.

#### 8-C. 알리·테무·징둥의 한국 직진출 영향
- **알리익스프레스**: 2025~2026 한국 1조 6천억 원 투자, 신세계 G마켓 합작법인 '그랜드오푸스홀딩' 설립.
- **테무**: 2024.2 '웨일코코리아' 한국 법인, 2026 경기도 물류센터·L2L(로컬투로컬) 모델.
- **징둥**: 인천·이천 물류센터(징동코리아), 2026 본격 운영.
- ***큐레이션* 셀러에게 미치는 영향**: *위탁 셀러*는 가격 압박이지만, *셀렉트 편집샵*은 알리·테무가 다루지 못하는 영역(셀렉 안목·세계관·디자인 큐레이션·라이프스타일 제안)에서 오히려 *반사이익*. 알리·테무는 객단가 5,000~15,000원의 초저가 시장이고, 큐레이션 객단가 30,000~80,000원과 시장이 분리.

#### 8-D. 일본 엔저 지속 → *일본 직소싱 황금 시대*
- 2026.1Q 100엔당 860원대 고착 (3년 전 대비 25% 저렴).
- 마키타·메트란 등 일본 제조 강소기업의 위기 = *한국 셀러에게는 매수 기회*.
- **권고**: SUPER DELIVERY를 *2026년 즉시 활용해야 할 1순위 채널*로 격상.

#### 8-E. 네이버 AI 쇼핑 에이전트
- 2026.2.26 베타 출시 (네이버플러스 스토어).
- 2026.4.27 AI탭 출시 (쇼핑·플레이스 통합).
- 2026 하반기 *광고주·사업주용* 에이전트 출시 예정 (최수연 대표 2026.4.30 컨콜 발표).
- **셀러 영향**: 단순 위탁 셀러는 AI 비교·요약으로 가격 압박 ↑↑. 셀렉트 셀러는 *상세페이지 스토리·리뷰 풍부한 자체 콘텐츠*가 AI 추천에 노출되어 오히려 유리할 가능성 — *단, AI가 해외 직수입 상품을 어떻게 다루는지는 베타 단계 데이터 부족, 하반기 모니터링 필수*.

#### 8-F. 동남아 친환경·핸드메이드 트렌드
- 베트남 라탄·발리 자수·태국 실크의 한국 셀렉트샵 노출 증가.
- 가격 경쟁력 + 디자인 차별화 + 친환경 스토리 3박자.

#### 8-G. 유럽 ESG 규제가 한국 수입에 미치는 영향
- **CBAM (탄소국경조정제도) 2026.1.1 본격 시행**: 철강·알루미늄·시멘트·비료·전기·수소 6개 품목, 연간 50톤 이하 면제 (2025.5 옴니버스 개정으로 90% 수입업체 면제). **셀렉트샵 카테고리(소품·문구·생활용품)는 사실상 무관**.
- **EUDR (산림 파괴 규제) 2026.12.30 시행** (대기업), 2027.6.30 (중소): 목재·팜유·커피·코코아·고무·축산물. **셀렉트샵 카테고리에 *일부 목재 상품(우드 트레이·프레임 등)*이 EU 수출 시 영향** → 단, 한국 내수만 한다면 무관.

## Caveats (해석상 주의사항)

1. **Accio Agent의 1,000만 MAU 수치 (2026.3)**는 알리바바 자체 발표(MIT Technology Review 인터뷰 인용)이며 제3자 검증된 수치가 아니다. 실제 활용 만족도는 사용자별 편차가 크다.
2. **목록통관 USD 150 한도**는 「관세법」 2026.7.1 시행 개정안에 따라 변경 가능성 있음 — 본 보고서는 2026년 5월 7일 시점 기준이며, 6개월 내 재확인 필수.
3. **윤석열 정부의 KC 직구 금지안 백지화**는 2025.4 정부 실각으로 사실상 무효화되었으나, 차기 정부 정책 방향에 따라 재추진 가능성 존재.
4. **1688 구매대행 수수료 5~10%**는 슈퍼레이블·굿게이트 등 한국 SaaS 평균치이며, 업체별·발주액별 큰 편차 있음.
5. **스마트스토어 등급 매출 기준** (씨앗 200만 미만, 새싹 200만+, 파워 800만+, 빅파워 4,000만+, 프리미엄 6억+/굿서비스, 플래티넘 100억+/굿서비스)은 *최근 3개월 누적* 기준이며 2022년 이후 큰 변화 없음. 단, 네이버는 정책 변경을 *수시 단행*하므로 매월 확인 필요.
6. **셀렉트샵 1인 운영자 성장 사례**는 공개 인터뷰·블로그 보고를 그대로 옮긴 것이며 매출 검증된 자료가 아니다. 사례에서 제시된 매출은 자체 보고치임을 감안할 것.
7. **CBAM·EUDR·미·중 관세** 등은 2026년 5월 시점 진행 중인 사안으로, 6개월 단위 재확인 필수.
8. **디자이너 강점**이 매출로 환산되는 데는 *콘텐츠 누적 시간(보통 12~18개월)*이 필요하며, 이 기간 동안은 일반 셀러보다 매출 성장이 *느릴 수 있음*. 이는 함정이 아니라 정상 패턴 — 누적 18개월 후에는 일반 셀러보다 마진율·재구매율에서 명확히 우월해진다.
9. **Faire/Ankorstore 역진출**은 빅파워+ 단계의 기회이지만, EU/미국 시장 진출에는 한국 디자인의 *현지 큐레이션 적합도*가 별개의 변수로 작용한다. 한국에서 잘 팔린다고 EU에서 잘 팔리는 것이 아님.
10. **본 보고서는 영구 baseline이지만 영원하지 않다.** 6개월마다 (1) 한국 관세법·KC 인증 정책, (2) 알리바바 Accio·네이버 AI 에이전트 진화, (3) 환율(특히 100엔당 / 위안당), (4) 알리·테무·징둥 한국 진출 진척도 — 4가지를 점검하고 본 보고서를 갱신할 것.

---

> **마지막 한 마디**: 디자이너 강점은 AI 시대에 *희소해진 자산*이다. 도매매 위탁 시기를 충분히 활용해 *큐레이션 헌법과 콘텐츠 자산*을 먼저 쌓아라. 해외 직소싱은 그 자산이 만들어 낸 매출이 *600만 원 임계값*을 넘기는 순간에 시작하는 게 정확한 타이밍이다. 그 전엔 준비만 하고, 그 후엔 망설이지 마라.
