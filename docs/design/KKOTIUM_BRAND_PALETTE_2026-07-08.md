# KKOTIUM 브랜드 팔레트 — 로고 기반 핑크-레드 (Phase 2b) (2026-07-08)

Authoring: DESKTOP(브랜드/UI). 구현=Code. 권위: 업로드 로고(브랜딩-로고-KKOTIUM 시안) 실색 추출 + DASHBOARD_SHELL_REDESIGN_RESEARCH(60/30/10·브랜드↔시맨틱 분리) + CONCEPT_DESIGN_TOKENS. 원칙 #218/#219/#220/#221. 비게이트(색상).
운영자 방향: 핑크+레드 메인·핑크 비중↑·연핑크·트렌디. 시맨틱색 트렌디 채도업. 추가 주색 대신 오브젝트로 다채로움.

================================================================
## 0. 로고 실색 추출 (Pillow)
================================================================
- 시그니처 레드: #F63B28 (심볼·엠블럼 주색·범위 #F63B28~#FD3E2B).
- 연핑크: #FCB4DC (배경·패턴·굿즈·뉴트로 로즈핑크·약간의 라벤더 틴트).
- 보조: 화이트·잉크블랙.

## 1. 확정 브랜드 팔레트 (CSS 변수)
================================================================
```
/* === 브랜드 시그니처 === */
--brand-red:        #F63B28;   /* 로고 레드 — CTA·primary·핵심 강조·활성·심볼 */
--brand-red-deep:   #D42818;   /* hover/active·깊은 레드 */
--brand-red-tint:   #FFE7E2;   /* 레드 배경 틴트(경고성 아님·강조 카드) */

/* === 핑크 (넓은 면적·지배색) === */
--brand-pink:       #FCB4DC;   /* 로고 연핑크 — 배지·서브강조·태그·하이라이트 */
--brand-pink-soft:  #FEE9F3;   /* 카드/섹션 배경 틴트 */
--brand-pink-wash:  #FFF5FA;   /* 페이지 최상위 캔버스 배경(거의 흰-핑크) */
--brand-pink-mid:   #FB7DBB;   /* (선택) 비비드 로즈 — 2차 강조·그래프·포인트 */

/* === 중립·텍스트 === */
--ink:              #2A1F1A;   /* 본문 텍스트(컨셉 유지) */
--ink-soft:         #6B5A52;   /* 보조 텍스트 */
--cream:            #FFF1E4;   /* 따뜻한 중립(컨셉·필요 시 핑크워시로 대체) */
--line:             #F0D9E4;   /* 핑크빛 헤어라인 보더 */

/* === 시맨틱 (브랜드와 분리·트렌디 채도업·운영자 확정) === */

--success-bg:       #E6FCF5;
--warning:          #FFB020;   /* 선명 앰버 — 밝고 명확(기존 머스타드 교체) */
--warning-bg:       #FFF4E0;
--danger:           #FF4757;   /* 코랄레드 — 브랜드핑크와 어울리는 생생한 위험색(브랜드레드#F63B28 주황끼와 색조 구분) */
--danger-bg:        #FFE9EC;
--info:             #4DABF7;   /* 스카이블루 — 중립 안내 */
--info-bg:          #E7F5FF;
```

## 1-b. 다채로움 전략 — "오브젝트로 곁들이기" (운영자 확정)
================================================================
★결정: 주 팔레트 = **핑크+레드 2색 고정** + 시맨틱. 추가 주색(3~4번째) 도입 안 함(브랜드 각인 약화·시선통제 붕괴 방지·#217).
- 다채로움은 **일러스트·마스코트·아이콘·스티커·장식 오브젝트에** 컨셉 보조톤을 점처럼:
  - 버터옐로우 #FFD93D · 라일락 #C8B6FF · 스카이 #BFE0E8 · 세이지 #6FA88F(컨셉 팔레트).
  - 꽃·나비·구름·꼬띠 일러스트가 다색이어도 **전체는 핑크-레드로 읽힘**(컨셉 Retro-Pop Garden 기법).
- UI 구조색(배경·버튼·텍스트·보더)은 핑크-레드-시맨틱만. 오브젝트/장식에서만 다색.
- 효과: 브랜드 각인(2색) + 트렌디·다채(디테일) + ADHD 집중(구조 절제) 동시.

================================================================
## 2. 색 분배 (60/30/10·핑크 지배)
================================================================
- 60% 넓은 배경 = 연핑크/워시(--brand-pink-wash 캔버스 + --brand-pink-soft 카드틴트).
- 30% 텍스트·구조 = --ink/--line.
- 10% 강조 = --brand-red(CTA·핵심·심볼) + --brand-pink(서브 배지).
- 핑크는 면적 지배, 레드는 포인트 시선 유도 = 트렌디 + 가독 + ADHD 시선통제.

================================================================

## 작업 유의사항/원칙 (신규)
================================================================
- **#220** 브랜드 색은 실제 브랜드 자산(로고)에서 추출 확정. 핑크(연·넓은 배경 지배)+레드(포인트·CTA).
- **#221** 시맨틱색도 트렌디 채도로(칙칙 지양)·의미 유지(성공 민트/경고 앰버/위험 코랄레드/정보 스카이)·위험색은 브랜드레드와 색조 구분. **주색은 2색(핑크+레드) 고정·다채로움은 일러스트/오브젝트로**(추가 주색 금지·브랜드 각인+집중 보존).
