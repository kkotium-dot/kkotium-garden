## 2. 어떻게 (핵심 전략 — 제 권고)
================================================================
### 원칙: "시그니처 모먼트, 균일 장식 아님"
프리미엄 브랜드는 장식을 온 화면에 도배하지 않고 **소수의 시그니처 순간**에 집중합니다. 꽃틔움도 3~4개 순간에 브랜드를 빛내고 작업 영역은 깔끔하게.

### 시그니처 모먼트 4곳
1. **대시보드 히어로(매출 영역)** = 감성 절정 — 분수대/정원 은유로 "오늘의 매출"을 브랜드 순간으로(#217 차분 기본·중간+에서 모션). "하루의 시작"의 정서적 앵커.
2. **빈 상태(empty state)** = 죽은 화면 → 브랜드 접점. 꼬띠 마스코트 + 정원 일러스트 + 친근 카피. 모든 빈 목록/초기 화면.
3. **꼬띠 마스코트 = 차분한 가이드** — 온보딩·빈상태·완료 축하에 등장(차분 기본은 정적·중간+는 표정/모션). 과하지 않게.
4. **온보딩/첫 방문** = 정원 메타포가 가장 빛나는 곳(새싹→수확 서사).

### 트렌디 승화 (mock 복제 아님)
- Claude Design mock은 맥시멀. → **2026 부티크 브랜드 감도**로 승화: 정제된 일러스트(과밀 아님)·넉넉한 여백·마스터 색 시스템·핑크-레드 브랜드.
- 작업 영역(테이블·폼·데이터)은 화이트+마스터 색 유지(집중). 브랜드 감성은 시그니처 모먼트에만.

### 장식강도 토글 = ADHD 안전망 (핵심 재확인)
- 컨셉 감성은 **중간/맥시멀 모드에** 살고, **차분(기본)은 깔끔**. → 대표님이 다이얼로 조절. "예쁨 on-demand, 기본은 집중."

### 가역 핑크 캔버스
- Phase D 착수 시 --page-bg를 var(--brand-pink-wash)로 flip → 연핑크 캔버스 복귀(컨셉 정합). 토큰 하나로.

## 3. 추가 아이디어 (제 의견·co-founder)
================================================================
- **꽃틔움 일러스트 시스템 자산화**: 꼬띠 포즈 세트 + 정원 오브젝트(꽃/나비/구름/분수) 재사용 라이브러리. 여기가 "다채로움(#221 오브젝트 보조톤)"이 사는 곳. Firefly/Adobe로 브랜드 일관 세트 생성 → Supabase 자산화.
- **마이크로 인터랙션(브랜드 맛)**: 성공 시 꽃 개화·꼬띠 반응 등 딜라이트. 단 prefers-reduced-motion 존중·차분 기본 정적.
- **빈 상태 = 브랜드 접점 총동원**: 모든 empty state를 꼬띠+일러스트+카피로(무미건조 화면 제거). 새싹셀러 정서 케어.
- **성취 축하 모먼트**: 첫 발행·첫 주문·마일스톤에 꽃 피는 축하(파워셀러 성장 서사 강화·리텐션).
- **로고 심볼 활용**: 로고의 꽃+사람 심볼을 로딩·빈상태·워터마크에 은은히(브랜드 각인).

## 4. Phase D 단계 (제안)
================================================================
- **D-1 대시보드 히어로**: 매출 영역 분수대/정원 모먼트(중간+ 모드)·가역 핑크 캔버스 옵션.
- **D-2 빈상태+꼬띠**: 전 빈상태에 마스코트+일러스트+카피.
- **D-3 일러스트 자산화**: 꼬띠/정원 오브젝트 세트 생성·배선.
- **D-4 마이크로 인터랙션+축하**: 성취 모먼트·딜라이트(모션 절제).
- **D-5 확장**: 다른 페이지에 시그니처 모먼트 선별 적용.
- 각 단계 장식강도 토글 정합·차분 기본·페이지별 검증.
