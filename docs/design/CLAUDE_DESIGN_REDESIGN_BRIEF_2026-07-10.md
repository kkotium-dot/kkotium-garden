## A. 기본 프롬프트 (구조·ADHD·브랜드)
================================================================
=== 프롬프트 시작 ===
너는 프리미엄 B2B SaaS 백오피스 전문 UI/UX 디자이너다. 네이버 스마트스토어 1인 셀러(성인 ADHD)를 위한 '꽃틔움 가든'의 대시보드·핵심 화면을 재설계하라. "극도로 깔끔+한눈에 들어옴"과 "레트로팝 가든 감성"을 동시에.

[사용자] 성인 ADHD. 빠른 선택·집중 최우선.
[ADHD 8대] 1)단일 초점 2)점진적 공개(결정≤3) 3)결정 부하 최소·기본값 4)명확한 다음 행동 5)일관성 6)기능 우선 라벨 7)시각 청킹(덩어리≤6) 8)모션 절제. →"3초 안에 뭘 할지 보이는가".
[브랜드] 레드#F63B28(CTA·활성)·연핑크#FCB4DC(포인트·헤더). 상태색 민트#12B886/앰버#FFB020/코랄#FF4757/스카이#4DABF7. 폰트 3역할(개성 디스플레이 제목·부드러운 본문·또렷한 데이터). 컨셉 레트로팝 가든·마스코트 꼬띠(빨간 튤립)·꼬끄(닥스훈트).
[대시보드 6청크] 상단 경보바(문제 시) / 존1 히어로60/40(오늘 할 일 단일 큐+KPI 4~5 큰숫자+스파크라인) / 존2 요약 3열(상품건강·받은편지함·관제탑·수치1+액션1+링크) / 존3 접힘.
[사이드바] 홈·상품·주문·고객·공급사·설정(기능명 우선·은유 보조).
[레이아웃] 여백·8px 그리드·부드러운 그림자·둥근 모서리·WCAG AA·reduced-motion·이모지 금지(아이콘).
[산출물] 1)대시보드 기본 2)빈상태(꼬띠) 3)사이드바 펼침 4)요약카드 3종 5)시장분석.

## B. ★정원 감성 보정 프롬프트 (이어서 붙여넣기·필수)
================================================================
=== 보정 시작 ===
구조는 유지하되 "정원 컨셉 감성"을 다시 입혀라. 지금은 너무 평범한 코퍼릿 SaaS다. 꽃틔움은 "레트로팝 가든 판타지" 브랜드다:
1. 정원 모티브를 "가장자리"에 풍부히: 헤더/섹션 구분선에 잎사귀·덩굴·작은 꽃, 카드 모서리 스캘롭 엣지, 와시테이프 라벨. (데이터/숫자 영역은 깔끔 유지)
2. 배경: 순백 대신 옅은 크림/피치(#FFF8F3)로 따뜻하게·섹션 연핑크(#FCB4DC) 카드 헤더 띠·성공/건강은 정원 그린 악센트.
3. 히어로(오늘 할 일 or 매출)를 "분수대/화단" 은유 시각화(물줄기=매출·꽃 피어남=성장)·숫자/목표 또렷이 병기.
4. 마스코트 꼬띠를 환영 헤더·빈상태에 등장(따뜻함)·꼬끄는 로딩/성공에.
5. 제목은 개성 디스플레이 폰트(Caprasimo류 둥글고 친근)로 크게.
6. 톤: "예쁜 정원 다이어리 + 프로 대시보드". 귀엽지만 신뢰감.
단, 매일 반복 데이터/폼/리스트는 절제(화이트+2색)·집중 방해 금지. 감성은 가장자리, 명료함은 중심.
=== 보정 끝 ===

## C. 사용 단계 + 후속
================================================================
1. claude.ai/design → A 프롬프트 → 시안 → B 보정 프롬프트 이어서 → 재생성.
2. 화면별 순차 요청·구체 수정 지시("핑크 더/덜"·"꼬띠 위치").
3. 완성 시안 → 스크린샷/export → 이 대화 공유 → Desktop 토큰/스펙 추출 → Code 구현 → prod 검증.
4. 대안: "Figma로 해줘" → Desktop이 Figma MCP로 직접 제작.

## 작업 유의사항/원칙 (신규)
================================================================
- **#234** 디자인 브리프/프롬프트는 "절제"만 강조하면 AI가 코퍼릿 미니멀로 해석해 브랜드 감성이 소실됨. **구조(뼈대)와 감성(살갗)을 분리해 둘 다 명시** — "깔끔한 정보 구조 + 브랜드 감성 표피". ADHD 명료함은 구조·시선 흐름에서 확보(감성 제거가 아님). 감성은 가장자리(헤더·빈상태·히어로·마스코트)에 풍부히, 데이터/폼/리스트는 절제. 재설계 AI 도구엔 "무엇을 절제하고 무엇을 화려하게"를 영역별로 명시.
