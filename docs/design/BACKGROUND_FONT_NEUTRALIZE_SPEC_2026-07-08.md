## 1. 배경 화이트 중립화 (당분간)
================================================================
- 현 prod: 캔버스 #fff5fa 연핑크워시 + radial-gradient. → **화이트(#FFFFFF or 아주 옅은 #FCFCFD)로 중립화.**
- radial-gradient 배경 제거(또는 거의 안 보이게).
- ★핑크는 **유지하되 역할 축소**: 넓은 배경 아님 → 배지·태그·포인트·활성 표시·카드 헤더 틴트로만. 레드=CTA/강조 그대로.
- 사유: Claude Design 전체 틀이 기능에 맞게 확정되기 전 연핑크 전면 적용은 후속 재작업 큼. 화이트 중립 기반 위에 컨셉 얹는 게 유연.
- 구현: 캔버스 토큰(--canvas/--brand-pink-wash 소비처)만 화이트로. 팔레트 토글은 유지(장식강도/팔레트 프리셋 그대로·기본만 화이트).
- ★가역 설계: 나중에 연핑크 복귀가 쉽게 — 단일 토큰(--page-bg 등) 교체로 화이트↔핑크 전환되게.

================================================================
## 2. 폰트 역할 분리 (독립 편집 구조)
================================================================
3역할 토큰으로 명확 분리(각각 독립 교체 가능):
```
--font-display:  "Caprasimo","Black Han Sans";  /* 트렌디/컨셉 — 제목·히어로·브랜딩 */
--font-body:     "Gowun Dodum", Pretendard;      /* 일반 본문 */
--font-readable: "Pretendard", system-ui;        /* ★가독성 전용 — 데이터·표·숫자·폼·KPI값·주문번호 등 정확·밀집 정보 */
```
- **가독성 전용(--font-readable)** = 수치·테이블·주문번호·가격·폼 라벨 등 "정확히 읽어야 하는" 영역. 트렌디 폰트가 가독성 해치면 안 되는 곳.
- Claude Design 재작업 시: --font-display/--font-body는 컨셉에 맞게 자유 교체, --font-readable은 가독성 고정.
- 적용: KPI 숫자·주문/상품 테이블·가격·폼 입력 → --font-readable. 제목·브랜딩 → --font-display. 나머지 → --font-body.

## 3. 마스코트 표정 갭 (차분 기본 완성·#62)
================================================================
- 현 prod: SectionHeader `kk-mascot-face`는 차분 숨김 ✓ / BUT 위젯 내부 표정(예: 돈심기 T_T·기타 ^_^ 위젯)은 노출.
- 수정: 위젯 임베드 마스코트 표정도 `.kk-mascot-face`(또는 동등 클래스) 부여 → 차분 기본 숨김에 포함. 프로젝트 전체 표정 요소 일괄(#62·단건 아님).
- 검증: 차분 기본에서 표정 0개(SectionHeader+위젯 모두).
