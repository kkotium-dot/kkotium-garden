## 0. 사용 원칙
- 디자인 작업 시작 전 이 문서를 도구(Claude Design 등)에 컨텍스트로 공급한다.
- 같은 프롬프트를 반복해도 컬러·간격·톤이 미묘하게 달라지는 "양산 불가" 문제를 이 토큰으로 고정한다.
- 상품 카테고리별 톤은 §6 변주 테이블을 따른다. 브랜드 코어(컬러·로고·마스코트)는 절대 변형 금지.
- 고객 노출 디자인(상세페이지/썸네일)은 한글 사용 정상. (앱 코드의 한글 리터럴 금지 규칙은 별개.)

## 1. 브랜드 아이덴티티
- **브랜드명**: 꽃틔움 KKOTIUM (스토어명 = 고객 노출 = "꽃틔움")
- **본질**: 큐레이션 셀렉트샵 — "골라주는 안목". 흔한 도매 상품을 감각적으로 재해석해 제시.
- **마스코트 꼬띠 (꼬띠)**: 빨간 튤립 + 핑크 카우걸 부츠. 위트 있고 따뜻한 안내자. 과용 금지(액센트로만).
- **보이스**: 전문적이되 따뜻하고 위트 있게. 과장·최상급 표현 금지(네이버 어뷰징 + #46 정직).

## 2. 컬러 토큰
| 역할 | 이름 | HEX | 용도 |
|---|---|---|---|
| 강조 (Accent) | Brand Red | `#E62310` | CTA 버튼, 핵심 강조, 로고. 전체의 약 10% |
| 보조 (Sub) | Brand Pink | `#FFCCEA` | 배경 포인트, 배지, 구분선. 약 15% |
| 베이스 (Base) | Neutral White/Gray | `#FFFFFF` / `#F7F7F8` / `#1A1A1A`(텍스트) | 여백·배경·본문. 약 75% |

**75 / 15 / 10 비율 원칙**: 베이스(화이트·뉴트럴) 75% · 핑크 15% · 레드 10%.
→ 상품 사진과 명화 색이 주인공이 되도록 브랜드색은 절제한다. 특히 명화 상품은 작품 색이 메인이므로 브랜드색을 더 줄인다.

**보조 뉴트럴**: 본문 `#1A1A1A`, 보조 텍스트 `#6B6B6B`, 라인 `#E5E5E5`, 카드 배경 `#FAFAFA`.

## 3. 타이포그래피
| 레벨 | 폰트 | 크기(모바일 기준) | 용도 |
|---|---|---|---|
| 대제목 (Hero) | Pretendard Bold / 세리프(아트 무드 시) | 28~36px | 후킹 카피 |
| 섹션 제목 | Pretendard SemiBold | 22~26px | 섹션 헤더 |
| 본문 | **Pretendard Regular** | **최소 16px / 행간 1.5** | 설명 텍스트 |
| 캡션 | Pretendard Regular | 13~14px | 작품명·출처·주석 |

- **한글 본문은 Pretendard 고정**. 아트 무드가 필요한 대제목·작품 캡션은 세리프 허용(명화 갤러리 감성).
- 모바일 본문 16px 미만 금지(가독성 + 약 75~80%가 모바일).
- 이모지 사용 금지 — 아이콘은 Lucide 스타일(선형, 미니멀).

---

## 4. 간격·레이아웃 토큰
- **상세페이지 캔버스 폭**: 제작 1000~1200px → export 860px (네이버 권장). 모바일 우선.
- **섹션 간 여백**: 80~120px (충분한 호흡 = 아트갤러리 무드의 핵심).
- **요소 간 간격 스케일**: 8 / 16 / 24 / 40 / 64px (8의 배수).
- **카드 radius**: 16px. 버튼 radius: 12px. 이미지 radius: 12~20px.
- **CTA 버튼**: 최소 48×48px(터치 타겟), Brand Red 배경 + 화이트 텍스트, 모바일 폭 90%.
- **그림자**: 절제(아트 무드). `0 4px 20px rgba(0,0,0,0.06)` 수준.

---

## 5. 컴포넌트 토큰 (재사용)
- **섹션 헤더**: 라벨(핑크 배지) + 대제목 + 1줄 서브카피.
- **스펙 카드**: 화이트 배경 + 라인 보더 + Lucide 아이콘 + 라벨/값 2단.
- **신뢰 배지**: 인증·리뷰·출처표기. 핑크 또는 뉴트럴, 레드 아님(레드는 CTA 전용).
- **작품 캡션**: 작가명 · 작품명 · 제작연도 · "Public Domain" 표기 (명화 상품 진정성).
- **CTA 블록**: 대형 레드 버튼 + 짧은 초대형 카피("지금 내 차에 들이기" 류).

## 6. 카테고리별 톤 변주 (코어는 고정, 무드만 조정)
| 카테고리 | 메인 무드 | 서브 강조 | 예시 상품 |
|---|---|---|---|
| 아로마/방향제/디퓨저 | 아트갤러리 + 감성 | 향·지속력·차량조화 | 명화 차량용 디퓨저 |
| 전통/도어벨/소품 | 한국적 미니멀·여백 | 소재·수공·정서 | 달항아리 도어벨 |
| 주방/생활용품 | 깔끔·실용·청결감 | 기능·위생·편의 | 아이스트레이 |
| 선물/기프트 | 따뜻함·설렘 | 패키지·메시지카드 | 공통 적용 |

규칙: 무드는 카테고리별로 바뀌어도 **컬러 토큰·타이포·간격·로고·꼬띠는 절대 고정**. 이게 "꽃틔움다움"을 만든다.

## 7. 절대 금지 (네이버 SEO + 브랜드 보호)
- 대표 썸네일에 텍스트·워터마크·외곽선·과도한 테두리 (네이버 검색 불이익·어뷰징 삭제).
- 최상급/과장 표현("세계 최고", "1위", "유일") — 진위성 게이트(#46) 차단 + 공정위 리스크.
- 타 브랜드 로고·명칭 도용. 사후 70년 미경과 작가 작품·생존 작가 화풍 무단 사용.
- 브랜드 컬러 임의 변경, 마스코트 변형, 이모지 삽입.
