## 3. ADHD 알림 문구 포맷 (표준)
**규칙**: 1알림=1행동·3초 판독·우선순위 이모지 선두·군더더기 0·딥링크 버튼·꼬띠 한마디 짧게.
```
[🔴 긴급] {상품명} 재고 10개 남음
→ 지금 할 일: 재입고 or 대체상품 등록
🌷 꼬띠: 놓치면 아까운 판매 기회예요!
[앱에서 열기]
```
- 우선순위 티어: 🔴긴급(재고 red·품절) / 🟠주의(재고 orange·마진·부활) / 🟢정보(추천·리포트).
- 🟢정보는 **아침 다이제스트 1회로 묶어** 알림 피로 방지. 🔴🟠만 실시간 개별.
- 제목=상황·화살표=행동·꼬띠=감성 한 줄·버튼=딥링크. (긴 문단 금지.)
