# 정원 대시보드 v6 진단 + v7 개선 (2026-07-10)

Authoring: DESKTOP(운영자 v6 PDF 실측). 권위: 운영자 확정(v5 정원 컨셉) + #237/#239. 상태: v6 검토·v7 프롬프트 준비.

## 0. v6 결과 (v5→v6 진척)
- v5 대비 **데이터 반투명 화이트 패널 추가**(오늘 할 일·매출·KPI에 흰 패널) → 부분 개선.
- 남은 문제(운영자 지적): (1) **중앙 분수가 안 보임** (2) 폰트/대비 가독성 여전히 약함.

## 1. 정밀 진단 (수석 디자이너)
운영자 지적 + Desktop 추가:
1. **분수 가려짐**: 매출 흰 패널이 분수 일러스트 "위"에 얹혀 랜드마크(분수)를 덮음 → 분수가 배경으로 밀림. 정원 중심 상징이 죽음.
2. **폰트 가독성**: 디스플레이(손글씨) 폰트가 제목뿐 아니라 곳곳에·연한 하늘/잔디 위 저대비 텍스트·라벨 작음. WCAG AA 미달 지점.
3. **전체 저대비**: 파스텔 일색이라 전경/배경 분리 약함. 숫자 외 텍스트가 묻힘.
4. **패널 처리 불일치**: 상단(매출·KPI)은 흰 패널 O, 하단 요약 3구획(온실/우체통/망루)은 진초록 배경+연한 텍스트 → 가독성·일관성 깨짐.
5. **위계 경쟁**: 상단 "오늘 할 일" 큰 보드 vs 중앙 매출 분수 둘 다 커서 시선 분산(ADHD 단일초점 약화).
6. **여백**: 상단 하늘 영역 과다(빈 공간)·세로 리듬 느슨.

## 2. v7 개선 방향
- **분수 = 명확한 랜드마크로**: 매출 패널이 분수를 덮지 말 것. 분수는 온전히 보이게 크게 그리고, 매출 숫자는 분수 "위 팻말/현판" 또는 분수 옆 깔끔한 패널로 분리(분수 가리지 않게). "분수+매출"이 하나의 또렷한 히어로.
- **폰트 규율**: 디스플레이(손글씨)는 **대제목 1곳만**. 나머지(라벨·숫자·본문) 전부 깔끔한 고대비 산세리프. 핵심 숫자 크게·다크(#2E2620)·tabular.
- **대비 강화**: 데이터가 얹히는 곳은 불투명도 높은 화이트/크림 패널(0.9+)·텍스트 다크. 파스텔 위 직접 텍스트 금지.
- **패널 일관성**: 하단 요약 3구획도 상단과 동일한 밝은 패널+다크 텍스트로 통일(진초록 배경 저대비 폐기).
- **단일 초점**: "오늘 할 일"과 "매출 분수" 중 하나만 최상위. 권장=오늘 할 일(액션) 최상위·매출은 바로 아래 히어로 지표.
- **여백 정리**: 상단 하늘 축소·세로 리듬 조여 1뷰포트 밀도.
- 유지: 정원 풍경·꼬띠·손그림·스캘롭·팔레트(밝은 빨강+핑크+크림).

## 3. v7 붙여넣기 프롬프트 (운영자용)
=== v7 시작 ===
v6 정원 컨셉 좋아, 유지해. 근데 두 가지가 문제야: (1) 중앙 분수가 흰 패널에 가려 안 보여. (2) 폰트/대비 가독성이 약해. v7로 이것만 고쳐줘(v6 보존):
1. 매출 흰 패널이 분수 일러스트를 덮지 않게 해. 분수는 온전히 크게 보이는 정원 중심 랜드마크로 그리고, 매출 숫자(₩4,287,500)는 분수 위에 얹힌 "나무 현판/팻말" 또는 분수 바로 옆 깔끔한 카드로 분리해서, 분수와 숫자가 둘 다 또렷하게.
2. 손글씨(디스플레이) 폰트는 맨 위 대제목 하나에만 써. 나머지 라벨·숫자·본문은 전부 깔끔한 고대비 산세리프로, 핵심 숫자는 크고 진한 다크색으로.
3. 파스텔 배경 위에 텍스트를 직접 올리지 말고, 데이터는 불투명도 높은(0.9+) 화이트/크림 패널 위에 얹어 또렷하게.
4. 하단 요약 3구획(온실/우체통/망루)도 진초록 배경 대신 위와 같은 밝은 패널+다크 텍스트로 통일해 가독성 맞춰.
5. 상단 하늘 여백을 줄이고 세로 간격을 조여 한 화면에 균형있게.
정원 풍경·꼬띠·손그림·스캘롭·밝은 빨강+핑크+크림은 그대로. 귀엽고 상쾌하되 숫자는 실무에서 확 읽히게.

## 작업 유의사항/원칙 (신규)
- **#243** 감성 배경 위 데이터 가독성은 "패널 불투명도(0.9+)+다크 고대비 산세리프 숫자+디스플레이는 대제목 1곳"이 규율. 브랜드 랜드마크(분수 등)는 데이터 패널이 "덮지" 말고 분리 배치(랜드마크·수치 둘 다 또렷). 한 화면 내 데이터 표기(패널·텍스트 대비)는 전 구획 일관(상단만 패널/하단은 저대비 금지). #239의 레이어 분리를 "랜드마크 비가림 + 전구획 일관"으로 확장.
