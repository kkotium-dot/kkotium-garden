## 2. 기능 감사 — 유지/이동/병합/제거 (27→핵심 6)

**A. 대시보드 유지 (지휘소 핵심·"지금 뭘·어떻게"):**

|---|---|
| KpiStrip(핵심지표4) | 유지·최상단(오늘매출·주문·정산·품절경보) |

**B. 전용 페이지로 이동 (분석·비상시 조회):**
| 위젯군 | 이동처 |

## 3. 프리미엄 B2B SaaS 레이아웃 (시선 흐름·비율)
================================================================
시선 흐름(Z/F 패턴·위→아래 우선순위):
```
[상단 얇은 경보 바] NaverHealth (조건부·문제 시만)
────────────────────────────────────
[존1·히어로 60%] 오늘 할 일 큐(TodayQueue)   |  [KPI 스택 40%] 핵심지표 4~5(세로)
   primary 액션 1개 강조                      |  오늘매출·주문·정산·품절·굿서비스
────────────────────────────────────
[존2·요약 카드 3열 균등] 상품건강 | 받은편지함 | 관제탑 요약
   각 카드=요약 수치 + "자세히" 링크(→전용 페이지)
────────────────────────────────────
[존3·접힘] 더보기(EventTimeline 등·기본 접힘)
```
- 비율: 존1 히어로가 시각 지배(60/40)·존2는 균등 3열·존3 접힘.
- 여백 넉넉·카드 그림자 절제·색은 화이트+마스터 상태색·브랜드 레드 CTA 1곳.
- 각 요약 카드는 "핵심 수치 1~2 + 액션 1"만(상세는 전용 페이지). 시선 분산 최소.
- 네비(사이드바 5그룹)에 /market·/growth·/control 추가.
