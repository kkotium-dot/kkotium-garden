## 1. ★systemic 발견 — 리셋이 수동 정정을 유실시킴
- 리셋(Phase 2b) = 저장 스냅샷 없어 **네이버 원본 재조회**로 복원.
- 명화의 네이버 원본 naver_origin = **"중국"(공급사 오등록)** → 운영자가 명화에 리셋 실행 시 **국산 정정이 중국으로 롤백됨**.
- 일반화: "수동으로 검증·정정한 필드(원산지·상품명 SEO·가격 등)"가 리셋으로 **공급사 원본값으로 유실**되는 전역 패턴(#62).
