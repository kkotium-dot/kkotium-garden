## 2. 대시보드 재구성 (IA-first·상단행동/중앙이해/하단탐색)
================================================================
**Phase 1 — IA 재편(최우선·최대효과)**:
- **최상단: "오늘 할 일" 단일 큐** — 현 3개 행동표면(오늘한가지+받은편지함+작업관제탑 개입점)을 **하나의 우선순위 To-do로 병합**. "지금 N건"+primary 버튼 1개. C-9 Operator Action Queue를 이 단일 큐로 표면화(신규 로직 아님·기존 개입점 통합).
- **상단 우측: KPI 4카드** — 오늘매출·신규주문·정산예정·품절경보. 큰숫자+델타화살표+스파크라인. (4=Cowan 상한.)
- **중앙: 이해** — 매출추이(분수대 위젯 여기·브랜드 존치)·최근 주문/리뷰 피드(색점).
- **하단: 탐색** — 스토어 진단·상세 링크(점진 공개·기본 접힘).
- 22카드 → 핵심 4~6개 + 나머지 점진 공개(더보기/2차화면).

**Phase 2 — 시각(IA 확정 후)**: 8pt 그리드·60/30/10·카드 규격(모서리·패딩·헤어라인)·시맨틱색 브랜드 분리(브랜드=꽃 핑크/코랄·성공=초록 별도).

================================================================
- 컨셉 프로토타입(branch `implement-kkottium-garden`·Vite·mock)은 **미감 레퍼런스**로 사용(팔레트·일러스트·분수대·마스코트·스캘롭 카드·와시테이프).
