## 1. 운영자 의도 (확정 — 재해석 금지)
================================================================
**"기능은 간결화(6청크·ADHD 워크플로우) + 대시보드 전체를 정원 지도처럼"** — 순서가 핵심:
1) 먼저 유료SW급으로 극도로 깔끔·선택과 집중되는 구조(이미 완료: 27→6청크·사이드바 6그룹).
2) 그 간결한 기능들을 **정원 요소로 변환**해 "들어갔을 때 귀엽고 상쾌한" 대시보드. (타 페이지는 절제 유지 가능·대시보드만큼은 정원.)
※ v2 실패 원인(#234): 구조만 남기고 감성 제거. 보정 프롬프트도 "장식 추가" 수준 — 운영자가 원한 건 **홀리스틱 정원 지도 은유**.

## 2. v1 디자인 DNA (스크린샷 실측 + 운영자 원본 프롬프트)
================================================================
운영자 원본 요청(박제): "빈티지+팝 컬러·스캘롭/웨이비·감각적 타이포·아날로그 스티커 레트로 감성 = 'Retro-Pop Garden Fantasy'를 B2B 대시보드에. 제1원칙 실무 워크플로우·데이터 가독성 최우선, 제2원칙 감성을 기능적으로 녹임."
- **중앙 분수대(수익 분수대)**: 대시보드 중앙 상단·"돈(물)이 얼마나 잘 흐르는지"·오늘 수익 지표+부드러운 애니메이션.
- **정원 지도 레이아웃**: 기능들=정원 요소(화단·꿀벌·나비·연못…)·정원을 가꾸며 스토어를 키우는 서사.
- **장식**: 손그림 일러스트(꽃·벌·나비)·스캘롭/물결 모서리·스티커/와시테이프·분수대와 화단 주변 자유 배치.
- **타이포**: studio dirt류 감각적 디스플레이 조합·과감한 배치.
- **컬러**: 빨강+핑크를 밝고 세련되게(부담스럽지 않게)·팔레트 토글 3~4.
- **위트**: 기능 상태에 반응하는 귀여운 오브제. 꼬띠 세계관(튤립 카우걸·꽃총 소싱·닥스훈트 꼬끄 배송·좀비부활소=만화적 세계).
- **인터랙티브**: 기능에 어울리는 인터랙션·풀 프로토타입(사이드바·페이지 클릭).

## 3. 융합 계획 (승인 시 Claude Design에 적용할 지시)
================================================================
**"v1의 정원 지도 감성 × 현재의 간결 6청크 구조"** — Claude Design 기존 프로젝트 대화에 Desktop이 직접 입력:
- v1(Kkottium Garden Dashboard.html)을 기준으로 삼되, 위젯 구성은 현행 간결 구조로 교체:
  · 중앙 상단 **수익 분수대**(오늘 매출+물 흐름 애니메이션) 유지·KPI 3~4는 분수대 주변 화분으로.
  · **오늘 할 일 = 메인 화단**(단일 primary·꼬띠가 가리킴)·나머지 할 일은 새싹 줄.
  · **요약 3구획**: 상품 건강=온실 / 받은 편지함=우체통(벌 배달) / 관제탑=망루. 각 수치1+액션1.
  · 시장/성장/관제 상세는 사이드바(정원 오솔길)로 이동 완료 — 대시보드엔 미배치.
  · 상태 반응 오브제: 주문↑=분수 세게, 품절=시든 꽃, 문의=벌 윙윙, 성공=나비.
- ADHD 8대(#233) 준수: 분수대·화단이 시선 위계 지배·결정≤3·3초 룰. 데이터 숫자는 또렷한 가독 폰트.
- 산출: 풀 프로토타입 1개(수정) → 운영자 확인 → 확정 시 Code 구현 인계.
