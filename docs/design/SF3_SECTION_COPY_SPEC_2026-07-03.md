## 3. 단계 증분
================================================================
- **SF-3a (섹션 카피 제안·읽기 전용)**: 각 섹션 슬롯에 소스에서 뽑은 **카피 제안** 표시(헤드라인+본문 초안). 복사 버튼. 영속 없음(SF-1식 안전 MVP). → 검증: 섹션별 제안 노출·hookPhrase 반영·소스 없으면 "생성 필요" 안내.
- **SF-3b (편집+영속)**: 제안을 편집 → 확정. 영속 방식 ★OPEN(§5) — 권고: 확정 카피를 detailContent/description 기존 배관에 합성(SF-4 내보내기와 연동) vs 경량 sectionCopy 필드. → 검증: 편집→재오픈 복원(영속 채택 시).
