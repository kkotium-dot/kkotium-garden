# 핸드오프 — 명화 디퓨저 무드 합성 레시피 (전상품 재사용 표준)

작성 2026-06-09 · 데스크톱. 권한 = REPRESENTATIVE_IMAGE_FINISHING_SYSTEM §7-1(추가이미지)·IMAGE_DETAIL_TWO_BRANCH_SYSTEM(Branch B mood). 목적: 공급사 본품을 감성 무드 장면에 자연스럽게 합성하는 **재사용 레시피**. 명화는 1호 사례.

## 0. 왜 이 문서
- 대표(흰배경 본품)는 §9 규격으로 별도. 본 레시피는 **추가이미지·상세 히어로용 무드 합성**.
- 이번 세션 Adobe MCP는 photo compositing/gen-fill/prompt-배경교체 미지원(#46) → 실행은 **Firefly 브라우저 반자동(#52)** 또는 Photoshop. 본 문서가 그 실행 스펙.

## 1. 소스 분석 — 첨부 Firefly 무드본 (교체 대상 장면)
- 광원: 좌측 창에서 들어오는 **따뜻한 오후 측광**(골든). 색온도 따뜻(~3500K 느낌).
- 그림자: 피사체 기준 **우측 하단으로 길고 부드럽게**. 하드엣지 아님.
- 배경: 크림색 커튼 보케 + 우측 원경 팜파스그라스 + 우드 테이블 상판(따뜻한 결).
- 구도: 본품이 좌측 1/3, 정면 약간 위 시점. 본품 높이가 프레임의 ~55%.

## 2. 교체 대상 → 명화 본품 스펙
교체: Firefly의 우드캡 리드디퓨저 → **명화 송풍구 디퓨저 실제 본품**(정사각 유리병 + 우드 볼캡 디퓨저 + 명화 라벨, 첨부 Image 2/4의 그 제품).
맞출 것:
- **비율/크기**: 교체되던 리드디퓨저와 동일 높이대(프레임 ~50~55%), 테이블 접지선 일치.
- **광원 정합**: 명화 라벨·유리·우드캡에 **좌측 따뜻한 하이라이트 + 우측 음영**. 유리 투명도에 배경 따뜻색 비침.
- **그림자**: 병 우측 하단으로 길고 부드러운 드롭섀도(불투명도 ~35%, 가우시안 블러). 테이블 반사 살짝.
- **색온도**: 장면과 동일 골든 톤으로 본품 화이트밸런스 보정(차갑게 튀지 않게).
- **접지**: 병 바닥-테이블 접촉부 어둡게(컨택트 섀도) → 떠 보이지 않게.

## 3. 실행 경로 A — Firefly 브라우저 반자동 (권장, #52)
1. 대표님: 로그인된 Adobe Firefly(또는 Photoshop 웹) 탭 오픈.
2. 데스크톱(Claude)이 구동: 본품 누끼(레퍼런스) 업로드 → generative-fill/reference 합성 프롬프트 입력 → 후보 생성·선택·폴링.
3. 대표님: 로그인·다운로드·비가역 클릭만.
- 프롬프트(영문, Firefly): `Place this exact car-fragrance diffuser bottle (square glass, wooden ball-cap, impressionist painting label) on a warm wooden table, soft golden afternoon side-light from the left, long soft shadow to the right, cream curtain bokeh background, pampas grass in the right background, photorealistic, matching light direction and color temperature, natural contact shadow.`
- 레퍼런스: 명화 본품 누끼(§4) 첨부.

## 4. 본품 누끼 소스 확보 (합성 입력)
- 1순위: 공급사 클린 단품컷(흰/밝은 배경) → `image_remove_background` → 투명 PNG.
- 명화 현재 대표(가죽 크롭)는 본품이 작고 배경 복잡 → 누끼 품질 낮음(비권장).
- 누끼 PNG가 합성·대표(§9) 양쪽의 재사용 입력.

## 4. 본품 누끼 소스 확보 (합성 입력) — ★전상품 표준(2026-06-09 세션2 대표 확정)
- **1순위 = 공급사 실촬영 단품 히어로컷**: 상세페이지 안의 크고 선명한 실사진(연출 배경 위 본품)에서 본품을 따기. 빛·그림자·재질이 살아있어 입체감 유지.
- **금지**: 작은 4종 변형 카드컷·텍스트 섞인 컷·저해상 썸네일 → 평면적·그래픽 느낌·잘림 발생(2026-06-09 세션2 1차 실패 원인).
- **본품 완전 포함**: 캡·병·라벨 전부 잘림 0(완전포함 원칙, T6 가드와 동일).
- 추출: 상세 원본(고해상)에서 히어로 영역 크롭 → image_remove_background(Adobe MCP, 복구됨) → 투명 PNG.
- 누끼 PNG가 합성·대표(§9) 양쪽의 재사용 입력.
- ★ 명화 실측: 풀해상 상세에 히어로컷 3종 존재(차량가죽·흰배경·리필동반). 3종 전부 누끼 산출완(cutout_A 들판소녀 / cutout_B 차량가죽 범선 / cutout_C 흰배경 들판마을). 작은 카드컷(170px)은 폐기.

## 5. 실행 경로 B — Photoshop 수동 (대안)
1. 무드 배경(Firefly 장면 or 자체 우드테이블 촬영) 레이어.
2. 본품 누끼 레이어 배치(비율 §2).
3. Camera Raw로 본품 색온도를 장면에 매치(따뜻).
4. 드롭섀도 + 컨택트섀도 수동(우측 하단·블러).
5. 전체 1회 컬러그레이딩(골든)로 통합. 1:1 또는 4:5 추가이미지 규격 export.

## 6. 산출물 용도
- 무드 합성본 = **추가이미지(2~9번) / 상세 S2 히어로**. 대표(1번) 아님(대표는 §9 흰배경).
- 전상품 재사용: 카테고리별 무드(디퓨저=골든 홈·차량 / 주방=밝은 키친 등)만 바꿔 동일 레시피.
