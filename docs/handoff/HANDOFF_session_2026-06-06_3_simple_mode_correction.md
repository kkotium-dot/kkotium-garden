## 0. ★ 핵심 정정 — 간편모드의 올바른 정의 (대표 직접 지적)

**오해(폐기)**: 업로드 사진을 누끼 따서 흰배경에 재합성 → 썸네일. (이건 너무 복잡, 간편이 아님)

**정답(확정)**: 상세페이지 퀄리티가 괜찮은 경우, **도매업체가 제공한 상세페이지 이미지 안에서 썸네일로 쓸 영역을 부분 크롭(1:1)해서 뽑아낸다.** 누끼·합성 불필요. "있는 걸 그대로 잘라 쓰기".

- 간편모드 = 도매 상세페이지 이미지 → 좋은 영역 1:1 크롭 → 규격화(1000px, JPEG). 끝.
- 누끼/합성은 상세페이지 퀄리티가 나쁠 때(보강·신규 모드)에만.
- 비유: 잘 차려진 뷔페(상세페이지)에서 덜어오기 vs 재료 사서 새로 요리(신규모드).

**3모드 재정의:**
| 모드 | 조건 | 썸네일 생성법 |
|---|---|---|
| 간편(70↑) | 상세페이지 퀄리티 양호 | 상세페이지 이미지에서 부분 1:1 크롭 |
| 보강(40~70) | 부분 부족 | 크롭 + 문구/이미지 보강 |
| 신규(40↓) | 소재 부실 | 누끼 + AI배경 합성(B안) |

## 2. 명화 간편모드 올바른 실전 — 다음 세션 진행

**필요한 것**: 명화 도매업체(도매매 65322245) 상세페이지 이미지. 현재 미확보(도매매 화보=사업자 로그인 필요, 실측됨). 대표가 상세페이지 이미지 제공 시 즉시 진행 가능.

**절차**:
1. 도매 상세페이지 이미지 확보(대표 제공 또는 로그인 후 다운로드).
2. 상세페이지 안에서 썸네일 적합 영역(깔끔한 단품/연출 컷) 식별.
3. 해당 영역 1:1 크롭(image_crop_and_resize 또는 crop_to_bounds) → 1000px, JPEG, 텍스트/워터마크 없는 영역 선택.
4. OCR 가드: 크롭본에 텍스트 있으면 텍스트 없는 영역 재선택.
5. 텍스트 SEO(상품명50자·카테고리50003356·속성·태그10) 초안 → 대표 승인 → 발행(비가역).
