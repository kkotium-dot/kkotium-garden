## 4-1. ★ 발행 워크플로우 — 준비도는 발행 승인이 아니다 (#307)

`준비도 %` / `발행 N/8` / `상품명 S등급`은 **필수 항목이 채워졌는지에 대한 체크섬**일 뿐, 그 내용이 팔릴 만한지에 대한 판단이 아니다.

```
정원 창고(미발행) → [준비도 체크 = 빠진 항목 없음]
   → ★씨앗심기 검수(운영자 필수): SEO 상품명·키워드·썸네일·상세
   → 발행 결정(운영자만)
```

- **미발행 → 발행 = 사람 검수 필수.** 발행됨 → 재활성화(일부 수정 + 튜닝 리셋) = 자동화 가능.
- 미발행 상품의 주 액션은 "발행"이 아니라 **"검수(씨앗심기로 이동)"**이다.
- 발행을 재촉하는 문구 금지. "발행 가능 N건 — 발행하러 가기" ✗ / **"검수 대기 N건 — 씨앗심기에서 확인"** ○
- **일괄 발행은 "검수 완료" 표시된 상품에만** 열린다. 준비도만으로 열면 검수 우회 유도다.
